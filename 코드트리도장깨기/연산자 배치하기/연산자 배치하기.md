# 1. 이해

```
[상황]
- n개의 정수가 순서대로 주어짐
- n-1개의 연산자를 정수 사이에 하나씩 배치
- 정수는 바꿀 수 없음
- 연산자는 (덧셈), (뺄셈), (곱셈) 세 가지 있음
- 연산자 간의 우선순위 무시하고 앞에서부터 걍 계산

[출력]
- 가능한 식의 (최솟값)과 (최댓값)을 출력하라

[입력]
- n (2 <= n <= 11)
- 숫자들 (1 <= 숫자 <= 100)
- 덧셈 뺄셈 곱셈 개수
```

# 2. 설계

```
- n이 11이 최대이므로 백트레킹 쌉가능
- 연산자를 섞어주면 된다
- 그냥 모든 경우의 수를 구해보자
- 우리가 할 수 있는건 0 1 2 3 4 5의 순열을 구하는 것
- 그럼 각 숫자에 연산자를 매칭해서 연산자를 섞을 수 있음

[설계]
1. 순열을 구한다
2. 만들어진 순열로 연산한다
	- 숫자랑 연산자랑 변환 배열로 인식해서 연산한다
3. 최소랑 최대 구한다
```
