# 1. 이해

```
[맵]
- N x N
- 각 격자에는 높이가 다른 리브로수가 있음
- 좌하단 4개의 칸에 초기 특수 영양제
- 0 : 씨앗
- 1이상 : 리브로수

[특수 영양제]
- 씨앗 -> 높이 1의 리브로수
- 리브로수 -> 높이+1

[이동 규칙]
- 이동 방향, 이동 칸 수
- 방향 : 1 ~ 8/우 우상 상 좌상 좌 좌하 하 우하
- *격자는 끝과 끝이 연결되어 있음

[1년간 성장]
1. 이동 규칙에 따라 이동
2. 특수 영양제 투입 -> 투입 후 사라짐
3. 특수 영양제가 투입된 리브로수는 대각선으로 인접한 방향에 높이가 1 이상인 리브로수가
있는 개수만큼 높이가 더 성장함. 단, 격자를 벗어난 리브로수는 세지 않음
* 높이를 일단 1을 증가시키고 나서 더 증가시키는거 고려해야 함

4. 특수 영양제를 투입한 리브로수를 제외하고 높이가 2이상인 리브로수는 높이 2를 베어서
특수 영양제를 사고, 해당 위치에 특수 영양제를 올려 놓음

[출력]
- 각 년도의 이동 규칙이 주어질 때, 해당 년수가 모두 지나고 난 뒤 남아있는 리브로수의 총합을
출력하라

[입력]
- 격자 크기 N (3 <= N <= 15), 키우는 년 수 M (1 <= M <= 100)
- 맵 정보
- M개의 이동 명령
```

# 2. 설계

```
[설계]
- 이동 명령 구조체
- 방향 배열 (우 우상 상 좌상 좌 좌하 하 우하)
- 함수
	1. 이동 규칙에 따라 특수 영양제가 이동하는 함수 구현 -> Move()
	2. 이동 이후 영양제를 투입하여 리브로수의 높이를 업데이트하는 함수 -> Inject()
	3. 특수 영양제를 맥인 르브로수를 '제외'하고, 높이가 2이상인 리브로수 높이 -2 업데이트하고
	특수 영양제 올려놓는 함수 구현 -> Get() O(N^2) 필요
	4. 위 함수들을 M번 반복 -> Simulation()

- 영양제 배열을 갖고 있어야 함
- 영양제를 주입한 위치는 visited처리해서 Get()함수를 구현하는데 사용해야 함
- 그리고 나서 visited는 다시 초기화되어야 함
```
