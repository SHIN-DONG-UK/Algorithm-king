# 1. 이해

```
[맵]
- n x n
- 1이상 6이하 숫자

[정육면체]
- 맵 위에 정육면체 하나 있음
- m번에 걸쳐 주사위를 계속 1칸씩 굴리게 됨
- 처음에는 항상 오른쪽으로 이동
- 주사위를 맞닿고 있는 칸의 상하좌우 인접한 같은 숫자의 합만큼 점수를 얻음
	- 이거 그냥 BFS
- 방향전환
	1. 아랫면의 숫자가 보드꺼보다 크다 -> 90도 시계방향
	2. 작다 -> 90도 반시계
	방향배열 시계방향으로 만들어 놔야할듯
	3. 격자판을 벗어나면 반대방향으로 돌고 한 칸 이동함
	
[출력]
- m번 진행하여 얻게 되는 점수의 총 합을 구하라

[입력]
- n : 격자 크기 (2 <= n <= 20)
- m : 굴리는 횟수 (1 <= m <= 1000)
- 맵
```

# 2. 설계

```
1. 방향배열 시계방향으로 선언해야겠다
2. 정육면체의 아랫면을 트래킹하는 함수를 만들어야겠다
3. 점수를 계산하는 BFS함수를 선언해야겠다
4. 모든 과정을 진행할 Simulation() 함수를 선언해야겠다
```
