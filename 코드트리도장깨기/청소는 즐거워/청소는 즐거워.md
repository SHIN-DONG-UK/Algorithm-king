# 1. 이해

```
[맵]
- n x n
- n은 홀수로 주어짐
- 처음 정가운데 먼지는 존재하지 않음
- 정가운데부터 시작하여 나선형으로 바닥 청소

[청소]
- 왼쪽 - 아래쪽 - 오른쪽 - 위쪽 순서로 청소
- n이 홀수이므로 항상 왼쪽 위에서 끝나게 됨
- 빗자루가 이동할 때마다 해당 칸에 먼지 사라지고 먼지가 이동하게 됨
	- a%는 원래 양에서 이외 이동한 먼지양을 뺀만큼임
	- 나머지는 퍼센티지로 이동하는데, 소숫점은 버림

[출력]
- 정중앙에서 시작해서 죄측 상단까지 ㄴ도달하는데 격자 밖으로 떨어진
먼지의 양 총합을 출력하라

[입력]
- n (3 <= n <= 499)
- 먼지 맵
```

# 2. 설계

```
[설계]
1. 정중앙에서 좌측 상단까지 이동하는 함수를 정의한다
	- 언제 꺾을까? 상하랑 좌우랑 짝꿍
	- 상하와 좌우를 1씩 번갈아 증가시키면서 이동한다
	
	(손으로 써보기)
	- 왼쪽 한 칸 이동
	- 아래쪽 한 칸 이동
	- 오른족 두 칸 이동
	- 위로 두 칸 이동
	- 왼쪽 세 칸 이동
	- 아래쪽 세 칸 이동
	- 오른쪽 네 칸 이동
	- 위쪽 다 섯칸 이동
	- 왼쪽 다 섯칸 이동
	- (0, 0)이다? 종료
	
2. 먼지 날리는 함수 정의한다
```
