[문제 요약]

- N x N 격자에서 택배 하차가 진행됨
- 택배 하차는 다음과 같은 방법으로 진행됨

```
1. 택배 투입

- 택배는 직사각형 모양으로, 직사각형 왼쪽 열의 위치 c, 가로 크기(w)와 세로 크기(h)으로 주어짐
- 각 택배는 택배 번호(k)를 가짐
- 투입한 택배들은 중력에 의해 하단으로 떨어짐
- 바닥에 닿거나 내려가던 중 다른 짐을 만나면 거기서 멈춤
- 택배 투입이 완료되었을 때, 어떠한 택배들도 공간을 벗어나지 않음이 보장됨

2. 택배 하차(좌측)

- 쌓인 택배 중에 잡고 왼쪽으로 이동했을 때 다른 택배와 부딪히지 않고 뺄 수 있는 택배를 먼저 하차
- 그러한 택배가 여러개일 경우, 택배 번호(k)가 작은 택배를 먼저 하차
- 해당 택배를 하차한 후, 공간에 남아있는 택배들 중 떨어질 수 있는 것들은 떨어짐

3. 택배 하차 (우측)

- 2의 과정을 우측에서도 진행

4. 반복

- 공간에 있는 택배를 모두 하차할 때까지 2, 3의 과정을 반복
```

- 하차되는 택배의 번호를 순서대로 출력하라

[제약 사항]

- 2 <= N <= 50 (맵 크기)
- 1 <= M <= 100 (M개의 택배 위치 정보)
- 1 <= k <= 100 (택배 번호)
- 1 <= w <= N (택배 가로 크기)
- 1 <= h <= N (택배 세로 크기)
- 1 <= c <= N (택배 왼쪽 열 위치)

[나이브한 생각]

- 이때까지 수행하던 1x1 크기의 정사각형에 대한 중력이 아닌, n x m 크기의 직사각형에 대해 중력을 적용하는 문제
- 사실 n x m 직사각형을 1x1 단위로 체크하면 될 것 같음
- 예를 들어, n x m 직사각형의 아랫변 정사각형만 충돌체크를 하면 된다
- 다만, 같이 움직이는 것 뿐
- 왼쪽 혹은 오른쪽으로 중력을 적용해야 할 때도 마찬가지

[설계]

1. 택배 투입 (초기 한 번)

- 택배의 왼쪽 열부터 가로크기만큼 레이저를 쏴본다
- 레이저를 쏜 것 중 가장 높은 위치 다음까지 택배를 이동시킨다
- 택배의 점유를 2차원 격자에 기록한다
- 택배를 번호로 검색했을 때 왼쪽 아래 모서리 좌표를 알고 있어야 하므로, 이를 기록해둔다
- 모든 택배를 위 순서로 수행한다

================== 여기서부터 모든 택배를 하차할 때까지 while loop ==============================

2. 택배 하차 (좌측)

- 택배 번호를 1부터 순회한다
- 만약 해당 택배가 이미 제거되었다면, 넘어간다
- 제거되지 않은 택배 중, 왼쪽으로 중력을 적용했을 때 걸리지 않으면 해당 택배를 제거 및 출력하고 반복을 종료한다.
- 택배 정렬 벡터를 행이 큰 순서대로 정렬한다
- 이후, 택배 정렬 벡터를 순회하며 생존한 택배에 대해 아래 방향으로 중력을 적용한다.


3. 택배 하차 (우측)

- 택배 번호를 1부터 순회한다
- 현재 택배가 이미 제거되었다면, 넘어간다
- 제거되지 않은 택배 중, 오른쪽으로 중력을 적용했을 때 걸리지 않으면 해당 택배를 제거 및 출력하고 반복을 종료한다.
- 택배 정렬 벡터를 행이 큰 순서대로 정렬한다
- 이후, 택배 정렬 벡터를 순회하며 생존한 택배에 대해 아래 방향으로 중력을 적용한다.


[자료 구조]

- 택배 구조체
	- 택배 번호
	- 왼쪽 아래 모서리 좌표
	- 너비
	- 높이
	- isRemoved

- 택배 멤버 함수
	- 아래 방향 레이저 -> return 옮길 번호
	- 왼쪽 레이저 -> true false
	- 오른쪽 레이저 -> true false
	- 2차원 격자에 기록
	- 2차원 격자에서 삭제

- 택배 메모리 풀
	- 번호 -> 택배 구조체

- 2차원 격자
	- 택배가 점유하고 있는 공간을 기록(택배 번호)

- 택배 정렬 벡터
	- 왼쪽 아래 모서리 좌표가 큰 순서대로 정렬하여 중력을 적용하기 위한 임시 벡터

- 남은 택배 수 cnt

[알고리즘]

1. (int) 아래 방향 레이저
- 왼쪽 모서리부터 오른쪽 끝 모서리까지 레이저를 쏴서 옮길 번호를 리턴하는 함수
- 옮길 번호 중 가장 작은 값을 리턴

2. (bool) 왼쪽 레이저
- 왼쪽 아래 모서리부터 왼쪽 위 모서리까지 레이저를 쏜다
- 만약 모서리 중 하나라도 다른 블록에 걸리면, return false
- 그렇지 않으면 return true

3. (bool) 오른쪽 레이저
- 오른쪽 아래 모서리부터 오른쪽 위 모서리까지 레이저를 쏜다
- 만약 모서리 중 하나라도 다른 블록에 걸리면, return false
- 그렇지 않으면 return true

4. (void) 2차원 격자에 기록
- 왼쪽 아래 모서리부터 블록 크기만큼 2차원 격자에 택배 번호를 기록

5. (void) 2차원 격자에서 삭제
- 왼쪽 아래 모서리부터 블록 크기만큼 2차원 격자에 0을 기록
- 그리고 isRemoved를 true로 설정
