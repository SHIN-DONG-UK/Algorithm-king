# 1. 이해

```
[상황]
- n개의 일이 주어질 때 이를 아침과 저녁으로 n/2개씩 나누어 처리
- 업무 간 상성 Pij가 주어짐

[출력]
- 아침과 저녁의 업무 강도 차 최솟값을 구하라

[입력]
- n (4 <= n <= 20)
- n은 2의 배수임
- Pij = 0 (i=j)
- 1 <= Pij <= 100 (i!=j)
```

# 2. 설계

```
1. 일단 아침에 할 업무와 저녁에 할 업무를 나눈다 -> 조합으로 나눔
2. 계산함수를 이용해서 강도의 합을 계산한다
	- 어떻게 할건데?
	- v1 = {1,2,3}, v2 = {4,5,6}
	- 2중 for문을 이용해서 i==j는 재끼고 01 02 10 12 20 21를 얻을 수 있음
	- 얘네들 다 대입해서 더해
3. v1벡터와 v2벡터를 어캐 얻을건데?
	- v1은 go하면서 push해놓고, v2는 visited되지 않은 애들 다 담아
```
