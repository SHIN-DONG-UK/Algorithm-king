# 1. 이해

```
[자율주행 자동차]
- N x M 크기의 트랙

[규칙]
1. 현재 방향을 기준으로 왼쪽 방향에 한 번도 간 적이 없다면 좌회전해서 왼쪽으로 1칸 전진
2. 왼쪽이 인도 or 이미 방문 -> 1번
3. 4방향 모두 전진 못하면 후진 -> 1번
4. 후진도 못하면 종료

[출력]
- 자율주행 자동차가 작동을 멈췄을 때 거쳐갔던 도로의 총 면적을 구하라

[입력]
- N, M (3 <= N, M <= 50)
- 자동차의 초기 위치, 방향 -> /북 : 0/동 : 1/남 : 2/서 : 3/
- 맵
	0 : 도로
	1 : 인도
- 도로 가장자리는 항상 인도임
```

# 2. 설계

```
- 주어진대로 시뮬레이션하는 함수를 구현하면 된다

[필요한거 정의]
1. 방향 배열 -> 북동남서 순서
2. 면적을 카운트하는 변수
3. simulation함수
```
