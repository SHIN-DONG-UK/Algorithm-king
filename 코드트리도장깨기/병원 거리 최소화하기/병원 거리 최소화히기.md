# 1. 이해

```
[도시]
- N x N 크기
- 사람, 병원, 빈 칸

[병원 거리]
- 각 사람에 대해 가장 가까운 병원까지의 거리
- |x1-x2| + |y1-y2|

[요구사항]
- m개의 병원만을 남겨두고 나머지를 폐업
- 병원거리 총 합이 최소가 되도록 폐업할 것

[출력]
- m개의 병원을 잘 골라 병원거리의 총 합을 최소화하라

[입력]
- N, M (2 <= N <= 50, 1 <= M <= 13)
- 병원의 수 ( m <= 병원의 수 <= 13)
- 도시
	0 : 빈칸
	1 : 사람
	2 : 병원
```

# 2. 설계

```
[설계]
1. 언제 최소가 될지 모른다 -> 다 해보면 무조건 답을 구할 수 있는데, 병원의 수가
 최대 13이라서 최대 2^13이니까 충분히 가능함
2. 병원의 개수가 k개라고 하면, k개 중 m개를 선택하는 모든 경우의 수를 구했다고 하자
3. 각 케이스에 대해 병원 거리를 다 구하고, 최솟값을 업데이트하면 된다

[필요함수]
1. k개 중 m개를 선택하는 함수
2. 해당 조합에서 병원거리를 구하는 함수
```
