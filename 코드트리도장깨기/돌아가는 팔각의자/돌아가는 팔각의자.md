# 1. 이해

```
[팔각 의자]
- 4개의 팔각 의자 존재
- 각 팔각 의자에는 N과 S가 있음
- 의자를 총 k번 회전하려고 함

[회전 요청]
- 회전시킬 의자의 번호와 방향
- n번째 의자가 회전하기 전 인접한 의자(n-1번째와 n+1번째)에 있던 의자에서 제일 가깝게 마주치는
두 명의 사람의 출신 지역이 다르다면 n번째 의자가 회전할 때 인접한 의자 또한 반대 방향으로 회전
- 회전 요청에 따라 의자를 회전한 후, 이로 인해 발생하는 모든 회전이 끝날 때까지 기다림
한 번 회전한 의자는 다시 회전하지 않음

[출력]
- 각 의자의 12시 방향에 앉아있는 남쪽 지방의 사람의 착석 여부를 출력하라
- 출력형식 : 1*s1 + 2*s2 + 4*s3 + 8*s4

[입력]
12시 방향부터 시계 방향 순서대로
- 첫 번째 줄 : 첫번째 의자
- 두 번째 줄 : 두번째 의자
- 세 번째 줄 : 세 번째 의자
- 네 번째 줄 : 네 번째 의자
	0 : 북
	1 : 남
- 회전 수 k (1 <= k <= 100)
- 명령들(k줄)
	n d (n : 1,2,3,4/ d : 1(시계) or -1(반시계))
```

# 2. 설계

```
1. n번째 의자와 n+1번째 의자에서 제일 가깝게 마주치는 두 자리를 알려주는 함수가 있어야 함
2. 이 함수에서 얻은 정보로 연쇄적으로 회전하는 함수 정의
3. k번 반복 후 0번째 인덱스를 보고 결과값 계산
```
