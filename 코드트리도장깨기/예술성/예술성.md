# 1. 이해

```
[그림]
- n x n
- 색깔 1 ~ 10
- 동일한 그룹 == 상하좌우 인접하고 같은 숫자

[예술 점수]
- 모든 그룹 쌍의 조화로움의 합으로 정의
- 그룹 a와 그룹 b의 조화로움 = (a칸 + b칸) x a숫자 x b숫자 x 'a와b가 맞닿아 있는 변의 수'

[초기 예술 점수]
- 예술 점수가 0이 아닌 모든 조합의 합

[회전 이후 예술 점수]
- 가운데 십자가 회전 -> 반시계 90도
- 나머지 네 구역 회전 -> 시계 90도

[출력]
- 초기 예술 점수, 1회전 예술 점수, 2회전 예술 점수, 3회전 예술 점수의 합을 출력하라

[입력]
- n (3 <= n <= 29)
- n은 무조건 홀수임
- 맵
```

# 2. 설계

```
1. 예술 점수 구하는 함수
	- 그룹을 나눠야 함 -> 몇 개 있는지 알아야 함
	- 걍 각 그룹 쌍마다 점수 해줘야 함
	- 그룹을 가리킬 수 있어야 함 -> 그룹을 표시해 놔야 함 -? a와 b가 맞닿아 있는 변의 수를 구하기 위함

2. 회전함수
	- 시계방향 90도
	- 반시계 90도
	- 가운데 십자만 회전
	- 십자로 나누어진 네 구역 회전
```
