# 1. 이해

```
[무빙워크]
- 길이가 n인 무빙워크
- 판의 개수 = 2n개
- 시계방향으로 회전
- 사람이 어떤 칸에 '올라가거나' '이동'하면 그 칸의 안전성은 즉시 1만큼 감소
- 안전성이 0인 칸은 올라설 수 없음

[안전성 테스트]
1. 무빙워크가 한 칸 회전
2. 가장 먼저 무빙워크에 올라간 사람부터 무빙워크가 회전하는 방향으로 한 칸 이동할 수 있으면 이동
만약 앞선 칸에 사람이 이미 있거나 안전성이 0이면 이동하지 않음
3. 1번 칸에 사람이 없고 안전성이 0이 아니라면 사람을 한 명 더 올림
4. 안전성이 0인 칸이 K개 이상이라면 과정을 종료, 아니면 반복
단, 1~3 과정 중 n번 칸 위치에 사람이 위치하면 그 사람은 즉시 내림
```

# 2. 설계

```
- 시키는대로 시뮬레이션 하면 된다

[필요 기능]
1. 안전성 개수를 파악하는 카운트 변수 -> zero_cnt
2. 안전성 배열을 돌리는 함수 -> RotateMovingWalk()
3. 사람 이동하는 함수 -> MovePeople()
	- 가장 먼저 탄 사람이 먼저 이동해야하니까, 역순으로 탐색하면서 사람을 옮기면 된다
	- 어차피 N칸에서 내리게 되어 있다
4. 사람 추가하는 함수 -> AddPeople()
```

# 3. 주의

```
- MovingWalk() 할 때 0인덱스 사람 없애줘야지!
- 0인덱스에 사람이 회전해서 올 수는 없다
- 그럼 뭐다? -> 무조건 0으로 refresh
```
