# 1. 이해

```
[시공의 돌풍]
- 항상 1번 열에 설치
- 두 칸 차지
- 일어나는 일
	1. 먼지가 인접한 4방향 칸으로 확산
		- 돌풍이 있거나 범위 벗어나면 컷
		- 확산되는 양은 원래 칸의 먼지/5
		- 확산된 먼지는 방에 있는 모든 먼지가 확산을 끝낸 다음에 해당 칸에 더해지게 됨
		
	2. 시공의 돌풍이 청소를 시작
		- 돌풍의 윗칸은 반시계 방향, 아랫칸은 시계 방향
		- 방향대로 한 칸씩 이동
		- 시공의 돌풍 안으로 들어간 먼지는 사라짐

[출력]
- 방의 정보가 주어질 떄, t초 후에 총 먼지의 양을 출력하라

[입력]
- n, m, t (6 <= n, m <= 50) (1 <= t <= 1000)
- 맵
	-1 : 돌풍
```

# 2. 설계

```
[설계]
- 돌풍 위치 파악 필요

1. 먼지가 인접한 4방향 칸으로 확산
	- 일단 맵을 2차원 벡터로 구성해야 겠다
	- 그래서 확산된 먼지를 뒤에 달아놓고, 모든 확산이 끝난 이후 각 칸의 벡터를 합해야 할듯
2. 시공의 돌풍이 청소를 시작
	- 먼지를 시프트하는 함수 정의
```
