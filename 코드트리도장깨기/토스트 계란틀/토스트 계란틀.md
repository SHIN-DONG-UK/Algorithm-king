# 1. 이해

```
[계란틀]
- N x N
- 각 칸에는 계란의 양

[규칙]
1. 계란의 양이 너무 차이나지 않게 하기 위해 하나의 선을 맞대고 있는 두 계란틀의 계란양 차이가
L이상 R이하라면 계란틀의 해당 선 분리
2. 모든 계란틀 검사 -> 1번 규칙에 해당하는 모든 계란 분리
3. 합치고 분리
	- 다 더한다음 등분
	- 소숫점은 버린다

[출력]
- 계란의 이동이 더 이상 필요 없을 때까지 반복할 때, 몇 번 계란 이동이 발생하는지 출력하라
```

# 2. 설계

```
1. 조건을 만족하는 모든 맞다은 계란틀을 구하기 위해, BFS함수를 정의해야 함
2. 이러한 BFS 함수가 있다면, 계란을 등분하는 Calc()함수가 필요
위 두 개의 함수만 있다면 이 문제는 해결할 수 있음

```
