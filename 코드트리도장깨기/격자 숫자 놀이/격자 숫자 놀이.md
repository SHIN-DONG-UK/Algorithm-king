# 1. 이해

```
[격자판]
- 3 x 3

[연산]
1. 행의 개수가 열의 개수보다 크거나 같은 경우
	- 모든 행에 대하여 정렬을 수행 -> 출현 빈도가 적은 순
	- 출현 횟수가 같은 경우 숫자가 작은 순
	- 정렬을 수행할 때 숫자와 해당하는 숫자의 출현 빈도 수를 옆에 추가

2. 열 개수가 행 개수보다 많은 경우
	- 모든 열에 대하여 1번을 수행
3. 행이나 열의 길이가 100을 넘어가는 경우에는 처음 100개의 격자를 제외하고 모두 버림
4. 변화된 격자의 크기는 가장 큰 길이를 기준으로 맞추고 나머지는 0으로 채움
```

# 2. 설계

```
[설계]
- 100 x 100 격자 필요
- 행의 개수와 열의 개수를 기록하는 변수 필요
- 해당 줄에서 숫자의 개수를 기록하는 DAT배열 필요
- 출현 빈도가 적은 순으로 정렬하고, 같은 수에 대해서는 숫자가 작은 순으로 정렬하는 커스텀 솥 필요
- 함수
	1. 열 또는 행 기준으로 [연산]을 수행하는 함수가 있다고 하자
		- 행의 개수가 열의 개수보다 같거나 크면,
			- 행 개수만큼 반복한다
			- 출현 빈도를 계산해 놓는다
			- 출현 빈도가 작은 순으로 정렬하고, 출현빈도가 같은 애들은 숫자가 작은 순으로 정렬하는
			커스텀 솥을 수행한다
			- 정렬해놓은 거를 이제 격자판에 넣기 시작하는데, 넣을 때 100이 넘어가면 멈춘다
			- 나머지를 0으로 채울필요 없다 배열은 이미 0으로 채워져 있으니까 
				-> 행의 개수를 업데이트하면된다
		- 열의 개수가 행의 개수보다 많으면 위 과정을 열 기준으로 한다
	
	2. A[r][c] 값이 k인지 매번 확인하는 함수가 필요
		- 시간 카운트
		- 위에서 만든 [연산]함수 수행
		- A[r][c] 값이 k인지 확인
			- 맞으면 멈춰
```
