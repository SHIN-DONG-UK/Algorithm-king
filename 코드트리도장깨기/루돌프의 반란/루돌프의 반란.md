# 1. 이해

```
[게임판]
- N x N
- 좌상단 (1, 1) -> 맵 마진 둘 필요성
- M개의 턴
	-> 매 턴마다 루돌프와 산타들이 한 번씩 움직임
	-> 루돌프 한 번 -> 1~P번 산타 이동
	-> 단, 기절해 있거나 격자 밖으로 빠져나간 산타들은 움직일 수 없음
- 두 칸 사이에 거리는 (r1-r2)^2 + (c1-c2)^2

[루돌프의 움직임]
- 상하좌우 대각선 돌진 가능
- 루돌프는 가장 가까운 산타를 향해 1칸 돌진 
	-> 단, 탈락하지 않은 산타
	-> 산타와 가장 가까워지는 방향으로 돌진
- 만약 가장 가까운 산타가 2명이라면, r좌표가 큰 -> c좌표가 큰

[산타의 움직임]
- 1번부터 P번까지 순서대로 움직임
- 기절했거나 이미 게임에서 탈락한 산타는 움직일 수 없음
- 산타는 루돌프에게 거리가 가장 가까워지는 방향으로 1칸 이동
- 다른 산타나 맵밖으로 이동 x
- 움직일 수 있는 칸이 없으면 이동 x
- 움직일 수 있는 칸이 있더라도 만약 루돌프로부터 가까워질 수 있는 방법이 없다면 이동x
- 산타는 상하좌우 이동
- 가장 가까워질 수 있는 방향이 여러 개라면, 우선순위 상-우-하-좌

[충돌]
- 산타와 루돌프가 같은 칸에 있으면 충돌 발생
1. 루돌프가 움직여서 충돌이 일어난 경우
	- 산타 C만큼 점수 + 산타는 루돌프가 이동해온 방향으로 C칸 밀림
2. 산타가 움직여서 충돌이 일어난 경우
	- 산타 D만큼 점수 + 자신이 이동해온 반대 방향으로 D칸 밀림
- 밀리는건 충돌 발생하지 않음
- 밀려난 위치가 게임판 밖이라면 산타 탈락
- 밀려난 칸에 다른 산타가 있다면 '상호작용' 발생

[상호작용]
- 착지하는 칸에서만 상호작용 발생 -> OK
- 다른 산타 1칸 밀림 + 연쇄작용 + 탈락 발생

[기절]
- '루돌프'와 충돌 후 기절함
- 기절한 산타는 한 턴 쉼
- 기절한 산타는 '밀려'날 수는 있음
- 루돌프는 기절한 산타를 돌진 대상으로 선택할 수 있음

[게임 종료]
- M번의 턴에 걸쳐 루돌프, 산타가 순서대로 움직인 이후 게임이 종료
- P명의 산타가 모두 탈락하면 게임 종료
- 매 턴 이후 아직 탈락하지 않은 산타들에게는 1점씩 추가 점수 부여
-----------------------------------------------------------------
[출력]
- 게임이 끝났을 때 각 산타가 획득한 점수를 출력하라

[입력]
- N, M, P, C, D
	N : 게임판 크기 MAX 50
	M : 게임 턴 수 MAX 1000
	P : 산타 수 MAX 30
	C : 루돌프 힘 MAX N
	D : 산타 힘 MAX N
- 초기 루돌프 위치 Rr Rc
- 산타 번호, 산타 위치
```

# 2. 설계

```
1. 루돌프 이동 함수
	- 가장 가까운 산타를 알아야 함
	- 가장 가까운 산타는 '거리'공식으로 찾으면 됨
	- 그러한 산타가 여러 명일 수 있으니, 2중 for문을 돌 때 열탐색 -> 행탐색 순으로 찾기
	- 단, 행과 열은 끝에서부터 시작
	- 단, 대각선에 산타 있으면 거기로 가야함
2. 산타 이동 함수
	- 산타를 관리하는 배열을 하나 둬야 겠다
	- 1번부터 P번까지 '순서대로' 이동해야 하니까
	- 또한 산타는 맵에도 기록되어 있어야 겠다
	- 맵과 배열을 동시에 사용해서 서로 업데이트 해야 한다
	- 마찬가지로 거리 식을 써서 가장 가까워지는 방향을 찾을 수 있다
		- np에 다른 산타가 있다 -> 컷
		- 맵 밖이다 -> 컷
		- 현재보다 거리가 더 멀다 -> 컷

3. 충돌 함수
	- 루돌프 이동
		- 충돌했다(루돌프 to 산타)
			a. 산타 점수 +C
			b. 산타 밀려남 +dy[dir]*C, +dx[dir]*C
				- 배열에도 반영
			c. 연쇄 반응 고려 +dy[dir], +dx[dir]
				- 배열에도 반영
	- 산타 이동
		- 충돌했다(산타 to 루돌프)
			a. 산타 점수 +D
			b. 산타 밀려남 +dy[dir]*D, +dx[dir]*D
				- 배열에도 반영
			c. 연쇄 반응 고려 +dy[dir], +dx[dir]
				- 배열에도 반영
	
3. Simulation() 함수
	- 남아 있는 산타 수 변수를 선언해놓고 산타가 튕겨 나갈때마다 update
	- cnt == 0이면 종료
	- Simulation()마지막에 아직 탈락하지 않은 산타들 +1점
```
