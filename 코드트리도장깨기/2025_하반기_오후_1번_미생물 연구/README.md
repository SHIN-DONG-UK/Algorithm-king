[문제 요약]

-  N x N 정사각형의 배양 용기에서 실험을 진행
- 좌측 하단이 (0,0) 우측 상단(N,N)
- 배양 용기에서 총 Q번의 실험을 진행
- 각 실험은 다음과 같은 순서로 진행됨

```
1. 미생물 투입
- 좌측 하단의 좌표가 (r1,c1)이고 우측 상단의 좌표가 (r2,c2)인 직사각형 영역에 미생물 투입
- 만약 영역 내에 다른 미생물이 존재한다면 새로 투입된 미생물이 그 영역 내 미생물을 잡아먹음
- 만약 무리 A에 의해 B가 둘 이상의 영역으로 나뉘게 될 경우, B는 배양 용기에서 모두 사라짐

2. 배양 용기 이동
- 모든 미생물을 새로운 배양 용기로 이동시킨다
- 새로운 배양 용기는 기존 배양 용기와 크기가 동일
- 이 과정은 기존 배양 용기에 미생물이 한 마리도 존재하지 않을 때까지 다음 작업이 반복됨

(1) 기존 배양 용기에 있는 무리 중 가장 넓은 영역의 무리를 하나 선택
	- 이러한 무리가 여러 개라면, 가장 먼저 투입된 미생물을 선택
(2) 선택된 무리를 새 배양 용기에 옮긴다
	- 기존 형태를 유지
	- 미생물 무리가 차지한 영역이 배양 용기의 범위를 벗어나면 안됨
	- 다른 미생물의 영역과 겹치면 안됨
	- 이 조건에서 최대한 x좌표가 작은 위치로 미생물을 옮긴다
	- 이러한 위치가 여러 개라면, 최대한 y좌표가 작은 위치로 오도록 옮긴다
	- 만약 어떤 곳에도 둘 수 없다면, 해당 무리는 소멸한다

3. 실험 결과 기록
- 실험 결과를 기록하는데, 다음과 같이 측정된다
- 미생물 무리 중 상하좌우로 맞닿은 면이 있는 무리끼리는 '인접한 무리'라고 표현하자
- 모든 '인접한 무리' 쌍을 확인한다
	- 이때 두 무리 A와 B가 맞닿은 면이 둘 이상이더라도 (A,B) 쌍은 한 번만 확인
- 두 무리를 A와 B라고 할 때, (A 영역의 넓이) x (B 영역의 넓이) 만큼 성과를 얻는다
- 확인한 모든 쌍의 성과를 더한 값이 이 실험의 결과
```

- N과 투입한 미생물 정보가 주어질 때, 각 실험의 결과를 출력하라

[제약 사항]

- 2 <= N <= 15
- 1 <= Q <= 50
- 0 <= r1 < r2 <= N
- 0 <= c1 < c2 <= N
- 모든 입력은 음이 아닌 정수

[나이브한 생각]

(1) 미생물 투입 후 무리가 두 개 이상으로 쪼개졌는지 찾아야 한다
	- 2차원 격자에서 BFS로 visited를 체크하면서 두 개 이상인지 확인해보면 된다
	- 즉, O(N^2)으로 가능하다
	- 한 번 찾는데 225번이고, Q가 최대 50이므로 시간은 충분하다

(2) 무리의 크기를 순서대로 알고 있어야 한다
	- 무리를 번호로 식별하자 == 무리 투입 순서
	- 무리에 대한 정보를 관리하자
		- 무리가 투입된 순서도 기록하고 있어야 함
		- isRemoved를 기록하고 있어야 함
	- 미생물 투입 시 무리의 크기를 업데이트하자

(3) 새 배양 용기에 빈공간을 찾아야 한다
	- N이 작으므로, 사실 Brute-Force하게 찾아도 된다

(4) '인접한 무리' 쌍을 찾을 수 있어야 한다
	- 이건 그냥 O(N^2)만에 가능하다
	- 배양 용기를 모두 돌면서 4방향을 탐색한다
	- 4방향을 탐색했을 때 처음 찾은 인접쌍이면, 계산한다
	- 인접쌍 방문 배열을 갖고 있어야 한다



[설계]

- N x N 격자에 미생물 무리 번호를 기록해둔다
- 미생물 무리 정보를 관리한다
	- 무리가 투입된 순서 == 무리 번호 (정적)
	- isRemoved (동적)
	- 무리 크기 (동적)
	- 왼쪽 아래 좌표 (동적)
	- 오른쪽 위 좌표 (동적)
- 미생물 무리 테이블을 만든다
- 정렬용 벡터를 둔다
- 방문 배열을 둔다
- 무리 번호 체크 카운트 배열을 둔다
- 인접쌍 체크용 bool 배열을 둔다

1. 미생물 투입 void input(int num, int y1, int x1, int y2, int x2)

- 미생물 무리 정보를 기록해둔다
- 배양 용기(2차원 격자)에 미생물 번호를 기록해둔다
- 정렬용 벡터에 미생물 포인터를 push_back한다
- 배양 용기를 돌면서, 쪼개진 무리가 있는지 확인한다
	- 방문 배열을 초기화한다
	- 현재 위치에 아무것도 없으면, 넘어간다
	- 현재 위치를 이미 방문했다면, 넘어간다
	- 현재 위치가 미생물이면, BFS로 방문 배열을 업데이트한다
	- 그리고 현재 무리 번호 체크 카운트 배열을 1 올린다
- 다시 배양 용기를 전체 순회한다
	- 카운트 배열의 값이 1보다 크면, BFS로 삭제한다

2. 배양 용기 이동

- 정렬용 벡터를 sort한다
	- 무리의 크기 오름차순
	- 그런 것이 여러 개라면, 무리 번호가 작은 순
- 정렬용 벡터를 순회한다
	- isRemoved == true이면, continue한다
	- 2차원 격자를 순회한다
		- 행 = N - 1 -> 0
		- 열 = 0 -> N -1
		- 현재 칸이 미생물이면, continue
		
		bool function {
		- 현재 미생물 무리 칸을 순회한다
			- 왼쪽 아래 -> 오른쪽 위
			- 기존 용기의 현재 칸이 미생물 번호와 다르면, 넘어가기
			- 새로운 용기의 현재 칸이 비어있지 않으면, 2차 loop 종료(return false)
		}
		- 만약 function이 true이면, 새로운 용기에 기록한다
		- 미생물 무리의 왼쪽 아래 좌표와 오른쪽 위 좌표를 업데이트한다
		- 2차원 격자 순회를 종료한다

	- 만약 기록한 적이 없으면, 해당 미생물 무리는 isRemoved = true이다


3. 실험 결과 기록

- sum 변수 필요
- 배양 용기를 순회한다
	- 현재 칸이 미생물일 때만 다음을 수행한다
	- 4방향을 탐색한다
	- 만약 다음 칸이 용기 밖이면, 넘어간다
	- 만약 다음 칸이 빈 칸이면, 넘어간다
	- 이미 인접쌍으로 기록했으면, 넘어간다
	- 만약 다음 칸이 다른 무리이면, 인접쌍 체크 배열을 true로 한다
	- 그리고 성과를 계산하여 sum에 합친다
- sum을 출력한다
