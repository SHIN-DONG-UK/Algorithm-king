# 1. 이해

```
[맵]
- n x m
- 불은 상하좌우 인접한 공간으로 모두 번짐
- 방화벽을 뚫을 수는 없음

[출력]
- 3개의 방화벽을 추가로 설치하여, 불이 퍼지지 않는 영역이 최대일 때의 그 영역의 크기를 출력하라

[입력]
- n, m (3 <= n,m <= 8)
- 총 불의 개수 (2 <= 총 불의 개수 <= 10)
- 총 빈칸의 개수 (3 <= 빈칸)
```

# 2. 설계

```
[설계]
- 백트레킹 문제 같다
- 그냥 빈 칸 추출하고 3개 선택하는 조합을 구한 후 불을 번지게 한다
- 번지게 한 이후 불 없는 영역 계산하고 업데이트
- 빈칸 vector 필요
- 불 vector 필요
- 맵 필요
- 불을 번지게 하는 BFS함수 필요
- 조합을 구하는 백트레킹 함수 필요
- 빈 칸을 세는 함수 필요
```
