[문제 요약]

- N x N 크기의 격자에 메두사가 살고 있다
- 도로는 0, 도로가 아닌 곳은 1

(메두사)
- 메두사의 집은 S, 공원은 E
- 메두사는 오직 도로만을 따라 최단 경로로 공원으로 이동함
- 메두사의 집과 공원은 항상 도로 위에 있고, 두 좌표는 항상 다르다

(전사)
- M명의 전사가 메두사를 잡으려고 함
- 각 전사의 초기 위치는 (ri, ci)
- 각 전사는 메두사를 향해 최단 경로로 이동
- 전사들은 도로와 비도로를 구분하지 않고 이동 가능
- 전사들의 초기 위치가 메두사의 집인 경우는 없음 -> 공원인 경우는 있음

(메두사와 전사들의 이동)
```
1. 메두사의 이동

- 메두사는 도로를 따라 한 칸 이동하며, 공원까지 최단 경로를 따름
- 메두사가 이동한 칸에 전사가 있을 경우 전사는 메두사에게 공격을 받고 사라짐
- 만약 집에서 공원까지 최단 경로가 여러 개라면, 상>하>좌>우 우선순위를 따름
- 메두사의 집으로부터 공원까지 도달하는 경로가 없을 수도 있음

2. 메두사의 시선

- 메두사는 상하좌우 하나의 방향을 선택해 바라본다
- 메두사는 바라보는 방향으로 90도의 시야각을 가지며, 시야각 범위 안에 있는 전사들을 볼 수 있음
- 메두사의 시야각 안에 들어와 있지만 다른 전사에 가려진 전사의 경우 메두사에게 보이지 않음
- 메두사가 본 전사들은 모두 돌로 변해 현재 턴에는 움직일 수 없으며, 이번 턴 이후 풀려남
- 만약 두 명 이상의 전사들이 같은 칸에 위치해있다면 해당 칸의 전사들은 모두 돌로 변함
- 이때 메두사는 상하좌우 중 전사를 가장 많이 볼 수 있는 방향을 바라봄
	- 같은 수의 전사를 볼 수 있는 방향이 여러 개라면, 상하좌우의 우선순위로 결정

3. 전사들의 이동

- 돌로 변하지 않은 전사들은 메두사를 향해 최대 두 칸까지 이동
- 전사들은 이동 중 같은 칸을 공유할 수 있음
- 첫 번째 이동
	- 메두사와의 거리를 줄일 수 있는 방향으로 한 칸 움직임
	- 이런 방향이 여러 개면 상하좌우 우선순위
	- 격자 바깥으로 나갈 수 없으며, 메두사의 시야에 들어오는 곳으로는 이동할 수 없음
- 두 번째 이동
	- 메두사와의 거리를 줄일 수 있는 방향으로 한 칸 더 이동
	- 이런 방향이 여러 개면 상하좌우 우선순위
	- 격자 바깥으로 나갈 수 없으며, 메두사의 시야에 들어오는 곳으로는 이동할 수 없음

4. 전사의 공격

- 메두사와 같은 칸에 도달한 전사는 메두사를 공격함
- 하지만 사라짐
```

- 위 네 단계가 반복되어 메두사가 공원에 도달할 때까지 매 턴마다 해당 턴에서 모든 전사가 이동한 거리의 합, 메두사로 인해 돌이 된 전사의 수, 메두사를 공격한 전사의 수를 출력하라
- 단, 메두사가 공원에 도착하는 턴에는 0을 출력하고 프로그램을 종료할 것

- 매 턴마다 현재 위치에서 공원까지 최단 경로를 구하고, 한 칸 이동하게 된다
- 공원을 초기위치로 한 플러드필을 한 번 수행하자
- 현재 메두사의 위치에 값이 기록되지 않았으면, 도달할 수 없는 것이므로 -1을 리턴하고 프로그램을 종료하자

- 현재 메두사의 위치에서 4방향을 한 번 탐색하자
- 4방향 에는 값이 기록되지 않은 칸과 기록된 칸이 있다
- 기록된 칸 중 가장 작은 값을 선택하자
- 단, 상하좌우 순서대로 찾자
- 먄약 이동한 칸이 공원이라면, 0을 출력하고 프로그램을 종료한다


2. 메두사의 시선

- 상하좌우 pVec[4] 클리어
- 상하좌우 방향 모두 시도해보자
- 3방향 BFS를 수행하며 전사 덩어리의 위치를 동적 배열에 저장해 놓는다
	- 상 -> 좌상, 상, 우상
	- 하 -> 좌하, 하, 우하
	- 좌 -> 좌상, 좌, 좌하
	- 우 -> 우상, 우, 우하
- 메두사의 시선을 기록하는 마스크에 방문 처리를 해준다
- 이제 전사 덩어리 동적 배열을 돌면서, 2방향 BFS를 수행하며 전사가 바라본 방향의 칸의 마스크 처리를 풀어준다
	- 상하좌우 방향에 일치한 전사들은 그 방향만 마스크를 풀어줌
	- 대각선 방향에 위치한 전사들은 (기본방향+사분면) 조합으로 2방향 BFS
		- 1사 -> 좌상 + 기본(대각선 왼쪽-좌, 대각선 오른쪽-상)
		- 2사 -> 우상 + 기본(대각선 왼쪽-상, 대각선 오른쪽-우)
		- 3사 -> 우하 + 기본(대각선 왼쪽-우, 대각선 오른쪽-하)
		- 4사 -> 좌하 + 기본(대각선 왼쪽-하, 대각선 오른쪽-좌)
