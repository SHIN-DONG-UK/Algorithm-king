# 1. 이해

```
[문제 요약]

- n x n의 나선형 미로속에 1번, 2번, 3번 몬스터들이 있다
- 격자 속 가운데 탑에서 몬스터들을 제거할거임
- 몬스터가 제거되는 과정
    1. 상하좌우 방향 중 (공격 방향, 공격 칸 수)만큼 공격하여 없앰
    2. 비어있는 공간만큼 몬스터는 앞으로 이동하여 빈 공간을 채움
    3. 몬스터의 종류가 4번 이상 반복하여 나오면 해당 몬스터들을 삭제
        - 더 이상 4번 이상 반복되지 않을때까지 삭제를 반복
    4. 삭제가 끝난 다음 몬스터를 차례대로 나열했을 때 같은 숫자끼리 짝을 지어준다. 이후 각각의 짝을 (총 개수, 숫자의 크기)로 바꾸어서 다시 미로 속에 집어넣는다

- 만약 새로 생긴 배열이 원래 격자 범위를 넘으면 무시
- 해당 과정이 끝나면 한 라운드가 끝남
- 1~3 과정에서 삭제되는 몬스터의 번호는 점수가 됨
- 모든 라운드가 끝난 후 플레이어가 얻게되는 점수를 출력하라

[제약 사항]

- 격자의 크기 n <= 25 (단, n은 홀수)
- 총 라운드 수 m <= 100
- 공격 방향 d 우하좌상
```

# 2. 설계

```
[설계]

- 몬스터들을 일차원 배열에서 관리해주면 되겠다

- 나선형으로 반복문을 돌 수 있어야 한다
    - get_monster 이런 함수를 둬서 1차원 배열로 가져온다

- 일차원 배열에서 빈칸이 있으면 왼쪽으로 땅기는 함수를 만들면 재활용할 수 있겠다
    - update_1darr 이런 함수를 둬서 1차원 배열을 정비한다

- 일차원 배열의 최대 길이는 25 * 25로 하면 되겠다

- 몬스터 종류가 4번 이상 반복되는걸 찾는 함수를 두자
    - 그리고 그걸 다 0으로 바꾼다
    - 그리고 return true 한다
    - 그런게 없으면 return false하도록 해서 반복을 종료할 수 있어야 한다

- 몬스터들을 짝짓는 함수도 만들어두자
    - 미로속에 집어넣자
```
