# 1. 이해

```
[격자판]
- n x n
- m개의 원자
	- 질량
	- 방향 : 상하좌우 대각선
	- 속력
	- 초기 위치 : x : 행, y : 열, 자연수
- 격자의 모든 행, 열은 각각 끝과 끝이 연결되어 있음

[시뮬레이션]
- 모든 원자는 1초가 지날 때마다 자신의 방향으로 속력만큼 이동
- 이동이 모두 끝난 뒤 하나의 칸에 2개 이상의 원자가 있는 경우에는 '합성'
	- 질량과 속력을 합
	- 4개의 원자로 나눠짐
	- 나누어진 원자들은 해당 칸에 위치 but 질량과 속력 방향은 다음과 같이 결정
		질량은 총질량의 /5
		속력은 속력합의 /개수
		방향은 모두 상하좌우중 하나거나 모두 대각선 중 하나였으면 상하좌우, 그렇지 않으면 대각선
	- 질량이 0인 원소는 소멸
- 이동 과정 중에 원자가 만나는 경우는 합성이 아님
	
	[출력]
	- k초가 될 때, 남아있는 원자의 질량 합을 구하라
	
	[입력]
	- n, m, k
		n : 격자 크기 4 <= n <= 50
		m : 원자 개수 0 <= m <= n^2
		k : 실험 시간 1 <= k <= 1000
	
	- m개의 원자 정보
		x, y, m, s, d
		x : 행 (1 <= x <= n)
		y : 열 (1 <= y <= n)
		m : 질량 (1 <= m <= 1000)
		s : 속도 (1 <= s <= 1000)
		d : 방향 (0 <= d <= 7)
		
```

# 2. 설계

```
[설계]
- 원자 구조체 정의
- 맵 필요
- 방향배열 : 상 우상 우 우하 하 좌하 좌 좌상
- 원자의 이동 함수가 필요 : 맵이 연결되어 있음!
- 2차원 맵 벡터를 만들어서 원자를 쌓아놓자
- 위 과정을 진행하는 Simulation()함수 필요
```
