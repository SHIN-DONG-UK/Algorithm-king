# 1. 이해

```
[맵]
- N x N크기의 도시
- 병원과 벽을 제외한 모든 지역에 바이러스
- M개의 병원을 골라 최대한 빨리 바이러스를 없애려고 함
- M개의 병원은 매 초마다 상하좌우로 인접한 지역 바이러스 제거

[출력]
- M개의 병원을 적절히 골라 바이러스를 전부 없애는데 걸리는 시간 최소

[입력]
- N, M ( 3 <= N <= 50, 1 <= M <= 10)
- 맵
	0 : 바이러스
	1 : 벽
	2 : 병원

```

# 2. 설계

```
[설계]
- 바이러스의 수를 카운트하는 변수를 가지고 있어야 겠다 -> 바이러스를 다 없앴는지 확인
- BFS를 도는데, M개의 노드에서 동시에 시작되어야 겠다 -> 처음 큐에 다 집어넣고 시작
- M개의 병원을 선택하는 경우의 수를 구하는 함수 필요 -> 백트레킹, 병원 리스트가 필요
```
