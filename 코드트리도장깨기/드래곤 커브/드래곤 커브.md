# 1. 이해

```
- 0차 드래곤 커브 == 길이가 1인 선분
- 1차 드래곤 커브 == 0차 드래곤 커브를 복제한 후 끝점을 기준으로 시계 방향 90도 회전
- n차 드래곤 커브 == n-1차 드래곤 커브의 끝점에 n-1차 드래곤 커브를 복제한 뒤 시계 방향으로
90도 회전시킨 뒤 연결한 도형

[출력]
- 100 x 100의 좌표평면에 n개의 드래곤 커브가 주어질 때 만들어지는 단위 정사각형의 개수를 구하라

[입력]
- 드래곤 커브 개수 n (1 <= n <= 20)
- 드래곤 커브 정보들
	x, y, d, g 
	x : 시작x
	y : 시작y
	d : 시작 방향
	g : 차수
- 방향 d : 0 1 2 3
					우상좌하
```

# 2. 설계

```
- n차 드래곤 커브를 만드는 함수가 있다고 하자
- 그럼 n번 반복하면서 드래곤 커브를 좌표평면에 찍고, 꼭짓점이 모두 1인 단위 정사각형 개수
카운트하면 된다

[설계]
- 좌표평면 : 100 x 100
- 방향배열 : 우상좌하
1. n차 드래곤 커브를 만드는 함수
	- 시계방향 90도 회전함수
	- 0차 커브부터 시작하면 되지 않을까?
	- n차 커브에서 n+1차 커브 만드는 방법
		a. 마지막 요소가 기준점
		b. 커브 배열에 반대로 가면서 기준점이랑 빼서 벡터 얻자
2. 단위 정사각형 카운트하는 함수
```
