# 1. 이해

```
[맵]
- n x n
	- 그루 수
	- 벽 정보

[제초제]
- k 범위 대각선
- 벽에 막힘

[나무의 성장과 억제]
1. 각 나무에 대해 인접한 네 칸 중 나무가 있는 칸 만큼 나무 성장
	-> 변화를 바로 적용해도 되는지 고민
2. 각 나무에 대해 인접한 네 칸 중 벽, 다른 나무, 제초제가 없는 칸에 번식을 진행
	- 이때 나무 개수에 번식 가능한 칸으로 나눈 만큼 각 칸에 번식
	- 나머지는 버림
	- 동시에 일어남
3. 각 칸 중 제초제를 뿌렸을 때 나무가 가장 많이 박멸되는 칸에 제초제를 뿌림
	- 나무가 없는 칸에 뿌리면 걍 거기서 끝남
	- 나무가 있는 칸에 뿌리면 4개의 대각선 방향으로 k칸만큼 전파
	- 단, 벽이나 빈칸이 가는 경로에 있으면 더이상 안감
	- 제초제가 뿌려진 칸에는 c년만큼 제초제가 남아있다가 c+1년째가 될 때 사라짐
	- 제초제가 뿌려진 곳에 다시 제초제가 뿌려지는 경우에는 새로 뿌져린 해로부터 다시 c년동안 유지
	- 박멸시키는 나무의 수가 동일한 칸이 있는 경우에는 행이 작은 순 -> 열이 작은 순
	
[출력]
- m년 동안 총 박멸한 나무의 그루 수를 출력하라

[입력]
- n, m, k, c
	n : 격자 크기 (5 <= n <= 20)
	m : 진행 년수 (1 <= m <= 1000)
	k : 제초제 확산 범위 (1 <= k <= 20)
	c : 제초제가 남아있는 년 수 (1 <= c <= 10)
- 맵 정보
	0 : 빈칸
	-1 : 벽
	자연수 : 나무
```

# 2. 설계

```
- 맵 필요
- 나무의 성장을 저장해놓는 다른 맵 필요
- 제초제 배열 필요

[시뮬레이션]
1. 나무를 찾아서 성장 시키는 함수 필요
2. 번식하는 함수 필요
3. 제초제를 뿌릴 위치를 찾고 뿌리는 함수 필요
4. 전체 과정을 진행할 반복함수 필요
```
