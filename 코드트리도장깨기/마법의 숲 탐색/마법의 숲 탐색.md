# 1. 이해

```
[맵]
- R x C (1행부터 R행)
- 동, 서, 남은 마법의 벽으로 막혀 있음
- 정령은 북쪽을 통해서만 숲에 들어올 수 있음
- K명의 정령은 각자 골렘을 타고 숲을 탐색함
- 골렘은 십자모양을 가지고 있음(5칸)
- 정령이 골렘을 탈 때는 아무방향이나 타도 되는데, 내릴때는 골렘도
출구가 따로 있음
- i번째 숲을 탐색하는 골렘은 숲의 가장 북쪽에서 시작해 골렘의 중앙이
ci열이 되도록 하는 위치에서 내려오기 시작함
- 초기 골렘의 출구는 di 방향에 위치해 있음

[골렘의 숲 탐색 우선순위]
1. 남쪽으로 한 칸 이동
2. 내려갈 때 바라보는 방향 양 팔이랑 머리가 끼워지도록
 비어있어야 내려갈 수 있음
3. 내려갈 수 없으면 서쪽 방향으로 회전하면서 내려감
4. 그래도 안되면 동쪽 방향으로 회전하면서 내려감
=> 골렘이 이동할 수 있는 가장 남쪽에 도달해 더 이상 이동할 수
없을 때까지 반복

[정령의 이동]
- 골렘이 다 내려오고 정령은 이동할 수 있음
- 정령은 가장 남쪽으로 이동할거임
- 다른 골렘을 타고 이동해도 됨
- 단, 골렘을 나올때는 출구로 나와야겠쥬?

[골렘의 초기화]
- 골렘이 최대한 남쪽으로 이동했지만 골렘의 몸 일부가 여전히 숲을
벗어난 상태라면, 해당 골렘을 포함해서 모든 골렘이 사라짐
- 그 다음 다른 골렘이 다시 숲 탐색
- 이 경우, 정령이 도달하는 위치를 답에 포함시키면 안됨

[출력]
- 골렘들이 숲에 진입함에 따라 각 정령들이 최종적으로 위치한
행의 총합을 구하라

[입력]
- R, C, K
	- R : 행 (5 <= R <= 70)
	- C : 열 (5 <= C <= 70)
	- K : 정령의 수 (1 <= K <= 1000)
- c, d
	- c : 골렘이 출발하는 열 (2 <= c <= C-1)
	- d : 골렘의 출구 방향
		- 0 : 북
		- 1 : 동
		- 2 : 남
		- 3 : 서
```
