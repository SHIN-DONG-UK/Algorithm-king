## 💡 BOJ 4307 - 개미[[문제 링크](https://www.acmicpc.net/problem/4307)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `애드 혹` |
| 사용 알고리즘 |  |
| 난이도 | `실버1` |

### ✅ 문제 설명
**[문제 요약]**

- 개미 여러 마리가 길이가 lcm인 막대 위에 있다
- 각 개미의 이동 속도는 모두 일정하며, 1cm/s이다
- 개미가 막대의 마지막까지 걸어간다면, 개미는 그 즉시 떨어진다
- 두 개미가 만나면, 방향을 반대로 바꾸어 걸어간다
- 가장 처음에 막대 상에서 개미 위치를 알고 있다
- 하지만 개미가 어느 방향으로 움직이는지는 알 수 없다
- 이때, 모든 개미가 땅으로 떨어질 때까지 가능한 시간 중 가장 빠른 시간과 느린 시간을 구하라

**[제약 사항]**

- 테스트 케이스로 이루어져 있음
- 각 테스트케이스마다 막대길이와 개미의 수 n이 주어진다
- n <= 100,000
- 막대 길이 l <= 1,000,000
- 0 <= 개미 위치 <= l

### 🔍 풀이 접근
- 목표 최솟값은 모든 개미가 최선의 방향으로 이동했을 때, 가장 마지막 개미의 이동 시간이다
   - 즉, 각 개미는 두 방향 중 최단 시간인 방향으로 이동한다
   - 그렇게 했을 때 각 개미의 이동 시간 중 최댓값이 목표 최솟값

- 목표 최댓값은 어떻게 구하지?
- Brute-Force하게 문제를 해결할 순 없다
- 그냥 양 끝에 있는 개미가 부딪힌다고 생각하면 되는거 아님?

### 💡 핵심 로직
- 이 문제는 각 개미가 서로 구분되지 않고, 똑같은 속도로 이동한다
- 그래서 개미들은 서로 통과한다고 생각해도 되는 논리적 합당함이 생긴다
- 결국 개미들이 통과한다고 인정하고 문제 푸는거임
