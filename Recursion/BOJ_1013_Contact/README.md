### ✅ 문제 설명
**[문제 요약]**

- 전파의 기본 단위는 { 0, 1 } 두 가지로 구성됨
- x+ ()는 임의의 개수(최소 1개) x의 반복으로 구성된 전파의 집합
- (xyx)+ ()는 괄호 내의 xyx의 반복으로 이루어진 전파의 집합
- 반복을 의미하는 +외에도 or를 의미하는 | 기호가 있다
- { x | y }는 x 혹은 y를 의미하는 것으로, { 0+ | 1+ }는 { 0, 1, 00, 11, 000, 111, ... }의 집합을 의미
- (100 | 11)+ = (100|11)(100|11)(100|11)...
- (100+1+ | 01)+ 이렇게 전파 패턴을 기록했는데, 다양한 전파 기록 중에서 이 패턴을 지니는 패턴인지 출력하라

**[제약 사항]**

- 테스트 케이스 T가 주어짐
- 각 문자열 길이는 1 <= N <= 200

---

### 🔍 풀이 접근
- (100+1+ | 01)+ = (100+1+|01)(100+1+|01)(100+1+|01)...
- 재귀적으로 문제를 접근할 수밖에 없다
- 첫 번째 패턴 -> 두 번째 패턴 -> ... 이렇게 순차적으로 탐색해나가면서 더 이상 가능하지 않으면 빠꾸하는 식으로 문제를 해결해야 한다
- 그러니까 이런 느낌이다
현재 패턴을 탐색한다 -> 01이네? 다음 패턴으로 넘어간다
현재 패턴을 탐색한다 -> 100+1+이네? 다음 패턴으로 넘어간다
현재 패턴이 둘 다 아니다 -> 빠꾸(return)친다

---

### 💡 핵심 로직
- 만약 1로 시작한다 -> 1다음 0이 최소 2개 이상 나온 이후, 1이 나와야 한다
- 1로 시작했으면 1로 끝나야 한다
- 0으로 시작했으면 무조건 01이 나와야 한다
- 0이 연속되는 건 100+1+ 에서 밖에 안되니까 그냥 연속한 0을 다 넘어가고 1을 찾는다
- 1이 연속되는 건 100+1+ 에서 밖에 안되니까 그냥 연속한 1을 다 넘어간다
-> 수정사항: 0은 다 털어도 되는데 1은 하나씩 털면서 다 해봐야 함
