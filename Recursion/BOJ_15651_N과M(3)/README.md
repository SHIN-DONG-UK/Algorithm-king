## 💡 BOJ 15651 - N과 M(3) [[문제 링크](https://www.acmicpc.net/problem/15651)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Brute-Force` |
| 사용 알고리즘 | `중복 순열` |
| 난이도 | 실버3 |

### ✅ 문제 설명
- 자연수 N과 M이 주어진다 (1 ≤ M ≤ N ≤ 8)

- 1부터 N까지 자연수 중에서 M개를 고른 수열인데, 중복이 허용된다

### 🔍 풀이 접근
- 재귀적으로 모든 노드를 탐색한다

### 💡 핵심 로직
- 순열 코드에서, 방문 처리를 빼면 된다

### 📌 배운 점
- 중복 순열 포맷을 이해했습니다.
