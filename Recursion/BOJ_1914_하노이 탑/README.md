### ✅ **문제 설명**`
**[문제 요약]**

- 하노이탑 문제

**[제약 사항]**

- 원판의 개수 N <= 100
- 옮긴 횟수 K를 출력하면 된다
- N <= 20이면 과정도 출력
- N > 20이면 과정은 생략

### 🔍 **풀이 접근**
- 1~b번째 원판을 옮기는 함수가 있다고 하자
- 예시에서 5번째 원판을 3번 장대에 옮기기 위해서는 1~4번째 원판을 2번 장대에 옮기고, 5번을 3번에 올렸다가 다시 1~4번째 원판을 3번에 올리면 된다
- 그럼 4번째 원판은? -> 1~3번째 원판을 2번 장대에 옮기고, 4번을 3번에 올렸다가 다시 1~3번째 원판을 3번에 올리면 된다
- 이거 재귀적으로 수행하면 되겠다

### 💡 **핵심 로직**
**[이동 순서]**
- hanoi(k, from, to, via)는 1번째 원판부터 k번째 원판을 from에서 to로 옮기는 함수다
- 만약 k가 0이 아니면, 다시 hanoi를 호출해야 한다
- 만약 k가 0이면, 더 이상 hanoi를 호출할 필요는 없다

**[큰 수 출력]**
- 문자열로 큰 수를 출력한다
- carry를 활용해 한 자리씩 곱하고 더한다
