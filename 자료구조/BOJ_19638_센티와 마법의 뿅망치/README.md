## 💡 BOJ 19638 - 센티와 마법의 뿅망치 [[문제 링크](https://www.acmicpc.net/problem/19638)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `자료구조` |
| 사용 자료구조 | `우선순위 큐` |
| 난이도 | `실버1` |

### ✅ 문제 설명
**[문제 요약]**

- 뿅망치를 맞으면 키가 절반이 됨
- 키가 1이면 뿅망치의 영향을 받지 않는다
- 매번 가장 키가 큰 거인 가운데 하나를 때리려고 함
- 이때 모든 거인의 키가 센티보다 작도록 할 수 있을 때 최소 횟수를 구하는 문제

**[제약 사항]**

- 거인의 나라 인구수 N <= 100,000
- 센티의 키 Hcenti, 거인의 키 <= 20억
- 뿅망치 횟수 제한 <= 100,000
- 뿅망치 다 썼는대도 거인의 키가 센티보다 작지 않으면, 그 중 가장 큰 키 출력

### 🔍 풀이 접근
- 항상 최댓값을 뽑아서 절반으로 줄이면 된다
- 만약 최댓값을 뽑았는데 센티보다 키가 작으면 그만두면 된다

### 💡 핵심 로직
- 우선순위 큐를 활용해 항상 최댓값에 O(logN)만에 접근하도록 한다

### 📌 배운 점
- 우선순위 큐를 활용해 최적해를 구하는 문제를 풀이했습니다.
