# 1. 이해

```
[맵]
- R x C (3 <= R, C <= 250)
- 영역 : 울타리를 지나지 않고 상하좌우로 이동할 수 있는 칸들
- 싸움 : 양의 수가 더 많으면 늑대를 몰아낼 수 있음, 그렇지 않으면 양은 다 잡아먹힘
[입력]
- R C
- 맵

[출력]
- 연산을 수행한 이후 양의 수, 늑대의 수
```

# 2. 설계

```
1. 맵(그래프)을 구성한다
2. 2중 for문을 돌면서 탐색을 시작할 수 있는 영역에서 dfs 탐색 시작한다
	- 탐색을 시작할 수 있는 곳?
		a. 방문하지 않은 곳
		b. 울타리가 아닌 곳
3. dfs돌면서 양의 수와 늑대의 수를 센다
4. 대소 비교해서 ans 변수에 반영한다
```
