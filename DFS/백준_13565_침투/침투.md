# 1. 이해

```
[맵 == 섬유 물질]
- M x N
- 위쪽 == 바깥쪽
- 아래쪽 == 안쪽
- 검은색 : 전류 차단
- 흰색 : 전류 흐름
- 전류는 섬유 물질 가장 바깥쪽 흰색 격자에 공급됨 -> 상하좌우 침투

[출력]
- 섬유 물질이 주어질 때, 전류가 안쪽으로 침투할 수 있는지 없는지 출력
- 안쪽에 전달되면 YES
- 그렇지 않으면 NO

[입력]
- M, N : (2 ≤ M, N ≤ 1000)
- 0 : 흰색
- 1 : 검은색
```

# 2. 설계

```
1. 맵(그래프)을 구성한다.
2. 가장 바깥쪽에 전류를 흘린다 == 시작 노드
	- 가장 바깥쪽은 0번 행 전부를 뜻한다
	- 그래서 0번행을 돌면서 다 시작해본다
3. 시작 노드에서부터 dfs탐색한다
	- dfs로 탐색하면서 가장 아래쪽에 닿으면 그냥 끝내버리거나 flag처리한다
```
