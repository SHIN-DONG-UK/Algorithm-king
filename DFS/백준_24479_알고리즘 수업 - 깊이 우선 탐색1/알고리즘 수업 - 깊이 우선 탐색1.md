# 1. 이해

```
- dfs코드 작성하면 됨

[입력]
- 정점의 수 N (5 <= N <= 100000)
- 간선의 수 M (1 <= M <= 200000)
- 시작 정점 R (1 <= R <= N)

[출력]
- 각 노드가 방문되는 순서를 출력하라
- 방문할 수 없는 노드는 0으로 출력
```

```
1. 그래프 구성

```
