# 1. 이해

```
- 음식물 존재
- 근처 음식물끼리 뭉치게 돼서 큰 음식물 쓰레기가 됨
- 제일 큰 음식물만 피해 가려고 함
- 상하좌우로 인접

[입력]
- 세로 : N (1 <= N <= 100)
- 가로 : M (1 <= M <= 100)
- 개수 : K (1 <= K <= NxM)
- (r, c) -> 왼쪽위가 기준&&(1,1)부터 시작

[출력]
- 가장 큰 음식물 쓰레기 크기 출력
```

# 2. 설계

```
1. 맵(그래프) 구성
2. dfs 탐색해서 인접한 쓰레기를 찾아나간다
3. 전체 맵을 2중 for로 돌면서 시작지점을 찾는다.
	- 확인한 음식물은 다시 확인하지 않도록 방문처리된 애들은 시작하지 않는다
	- 가장 큰 크기를 업데이트 한다
```
