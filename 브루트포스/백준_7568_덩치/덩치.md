# 1. 이해

```
[덩치]
- 몸무게 x, 키 y : (x, y)
- A의 덩치가 B의 덩치보다 크다 : A(x) > B(x) && A(y) > B(y)
- 덩치를 비교할 수 없다 : 
	A(x) > B(x) && A(y) < B(y) || A(x) < B(x) && A(y) > B(y)

[덩치의 등수]
- 자신의 덩치보다 큰 사람의 수
- 등수가 같은 사람이 여러 명일 수 있음

[출력]
- 나열된 사람의 덩치 등수를 구해서 출려하라

[입력]
- 전체 사람의 수 : N (2 ≤ N ≤ 50)
- 몸무게, 키 : x, y (10≤ x,y ≤ 200)
```

# 2. 설계

```
1. 이거 N이 50이하니까 2중 for문 해도 됨 (4중 for문까지 가능)
2. 그럼 걍 2중 돌면서 다 비교해주면 됨
```
