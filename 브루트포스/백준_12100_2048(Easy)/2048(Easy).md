# 1. 이해

```
[게임 설명]
- N x N 보드
- 전체 블록을 상하좌우 방향 중 하나로 이동시킬 것
- 같은 수가 충돌하면 두 블록은 하나로 합쳐짐
- 1회 이동에서 한 번 합쳐진 블록은 또 합쳐지지 않음

[출력]
- 최대 5번 이동시켜서 얻을 수 있는 가장 큰 블록을 출력하라

[입력]
- N
- 초기 보드 상태
```

# 2. 설계

```
- 먼저 블록이 합쳐지는 구현이 필요하다.
- 그리고 가장 큰 블록을 찾으라고 하는데, 모든 방향을 다 해본다음 최댓값을 찾는 Brute Force
알고리즘으로 구현하면 되겠다
- 모든 경우의 수를 다 해보기 위해, 중복순열을 구현하고 해당하는 커맨드대로 블록을 이동시킨다음
최댓값을 갱신하면 되겠다
```
