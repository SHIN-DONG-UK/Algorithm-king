# 1. 이해

```
[이해]
- N개의 마을을 두 개의 지역구로 나눌거다
- 두 지역구의 유권자 수 차이가 최소가 되도록 지역을 분리할거다
- 근데 마을끼리 연결되어 있지 않으면 안된다
- 모든 마을은 반드시 지역구 중 하나에 포함되어야 함

[출력]
- 두 개의 지역구로 나누었을 때, 유권자 수 차이 최소를 출력하라

[입력]
- 마을 수 N (MAX 10)
- i번째 마을의 유권자 수 Pi가 순서대로 주어진다
- N개의 줄에 마을 정보가 주어짐
```

# 2. 설계

```
1. A 지역구와 B 지역구를 어떻게 나눌까?
- 그냥 냅다 2^N 하면 되잖소(MAX 10이므로)
- 그리고 나서 이놈들이 연결되어 있는지 검사해주고 최댓값 갱신하면 되잖소
- 재귀함수로 모든 지역구 나누기 쌉가능

2. 연결 검사
- 지역구를 나누긴 했는데, 노드들이 서로 연결되어 있는지 검사가 필요
- 하나의 노드에서 다른 모든 노드로 갈 수 있어야 함
	-> BFS로 검사하자
```
