## 💡 BOJ 11047 - 동전0 [[문제 링크](https://www.acmicpc.net/problem/11047)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | 거스름돈 문제 |
| 사용 알고리즘 | Greedy |
| 난이도 | 실버4 |

### ✅ 문제 설명
- 준규가 가지고 있는 동전 N종류

- 그 가치의 합을 K로 만들려고 함

- 이때 필요한 동전 개수의 최솟값을 구하라

### 🔍 풀이 접근
- 거스름돈 문제

- 동전 가치가 서로 배수 관계라서 최대한 큰 단위를 주는게 최적해임

### 💡 핵심 로직
```cpp
void simulation(){
    for(int i=N-1; i>=0; i--){
        ans += K / coins[i];
        K = K % coins[i];
    }
}

```
