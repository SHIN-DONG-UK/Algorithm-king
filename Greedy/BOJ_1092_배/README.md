## 💡 BOJ 1092 - 배 [[문제 링크](https://www.acmicpc.net/problem/1092)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Greedy` |
| 사용 알고리즘 | `정렬` |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 항구에는 크레인이 N대 있고, 1분에 박스를 하나씩 배에 실을 수 있다
- 각 크레인은 무게 제한이 있다
- 이것보다 무거운 박스는 크레인으로 움직일 수 없다
- 모든 박스를 배로 옮기는데 드는 시간의 최솟값을 구하라

**[제약 사항]**

- N <= 50
- 무게 제한 <= 1,000,000
- 박스의 수 M <= 10,000
- 박스의 무게 <= 1,000,000
- 모든 박스를 배로 옮길 수 없으면 -1을 출력

### 🔍 풀이 접근

- N개의 크레인을 돌면서 자기가 옮길 수 있는 가장 큰 박스를 항상 옮긴다고 생각해보자
- 그리디하게 큰거부터 차례대로 박스를 옮기면 된다

### 💡 핵심 로직
- 박스 무게를 내림차순으로 정렬한다
- 크레인 무게 제한도 내림차순으로 정렬한다
- 무게 제한이 큰 놈부터 박스를 옮긴다
- 박스를 돌면서 처음으로 내 무게 제한보다 낮으면, 걔를 옮긴다(방문처리한다)
- 만약 현재 박스가 방문처리 되었으면, 그냥 넘어간다
- 이렇게 한 번 모든 크레인을 돌았으면, 횟수를 카운트한다
