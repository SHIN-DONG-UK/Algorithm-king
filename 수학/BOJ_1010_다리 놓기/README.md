## 💡 BOJ 1010 - 다리 놓기[[문제 링크](https://www.acmicpc.net/problem/1010)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `수학` |
| 사용 알고리즘 | `조합` |
| 난이도 | `실버5` |

### ✅ 문제 설명
**[문제 요약]**

- 강을 건너는 다리를 짓기 적합한 곳을 사이트라고 한다
- 강 서쪽에는 N개의 사이트가 있고, 동쪽에는 M개의 사이트가 있다
- N <= M
- 서쪽 사이트 개수만큼 다리를 지으려고 한다
- 다리끼리는 서로 겹칠 수 없다고 할 때, 다리를 지을 수 있는 경우의 수를 구하라

**[제약 사항]**

- T개의 테스트 케이스로 이루어짐
- N <= M < 30

### 🔍 풀이 접근

- 서로 겹칠 수 없다는 건 순서가 결정되어 있다는 것과 동치이다
- 따라서 mCn을 구하는 것과 같은 문제

### 💡 핵심 로직

- mCn을 구하는데, overflow를 방지하기 위해 곱셈과 나눈셈을 한 번에 적용한다

### 📌 배운 점
- overflow에 대한 문제 인식을 했습니다.
- "파스칼 삼각형이 정확히 조합의 DP적 표현"이라는 사실을 이해했습니다.
