## 💡 BOJ 1052 - 물병[[문제 링크](https://www.acmicpc.net/problem/1052)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `수학` |
| 사용 알고리즘 | `비트마스킹` |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- N개의 물병이 있다
- 각 물병에는 무한대로 물을 부을 수 있다
- 처음에 모든 병에 1리터의 물이 있다
- 물병을 다른 장소에 옮기려고 한다
- 한 번에 K개의 물병을 옮길 수 있다
- 물병의 물을 재분배해 K개를 넘지 않는 비어있지 않은 물병을 만들려고 한다
- 물은 다음과 같이 재분배한다
  - 먼저 같은 양의 물이 들어있는 물병 두 개를 고른다. 그 다음에 한 개의 물병에 다른 한 쪽에 있는 물을 모두 붓는다. 이 방법을 필요한 만큼 계속 한다.
- 이런 제약 때문에, N개로 K개를 넘지않는 비어있지 않은 물병을 만드는 것이 불가능할 수 있다
- 다행히도, 새로운 물병을 살 수 있다 (1리터 짜리)
- 상점에서 사야하는 물병의 최솟값을 구하라

**[제약 사항]**

- 1 <= N <= 10,000,000
- 1 <= K <= 1,000
- 정답이 없을 경우 -1 출력

### 🔍 풀이 접근

- 물병은 2의 거듭제곱으로 합쳐진다
- N으로 2의 거듭제곱을 K개 이하로 만들 때, 부족한 수를 구하는 문제임

### 💡 핵심 로직

- N을 이진수로 바꾼다
- 1의 개수를 센다 (cnt)
- K >= cnt -> 0출력
- K < cnt
  - 이진수에서 상위 1을 K-1개 지운다
  - 그 다음 상위 1부터 끝까지의 이진수에서, 0 자리 수 + 1이 정답이다
  - 이렇게 하는게 제일 싸다
