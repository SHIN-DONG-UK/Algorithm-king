## 💡 BOJ 1193 - 분수찾기[[문제 링크](https://www.acmicpc.net/problem/1193)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `수학` |
| 사용 알고리즘 | `수열의 합 공식` |
| 난이도 | `실버5` |

### ✅ 문제 설명
**[문제 요약]**

- 무한히 큰 배열에 다음과 같이 분수들이 적혀있다
1/1 1/2 1/3 1/4 1/5 ...
2/1 2/2 2/3 2/4 ...
3/1 3/2 3/3 ...
4/1 4/2 ...
5/1 ...
...

- 지그재그 순서로 차례대로 1번, 2번, 3번, 4번, 5번... 분수라고 하자
- x가 주어졌을 때 x번째 분수를 구하라

**[제약 사항]**

- x <= 10,000,000

### 🔍 풀이 접근
- 순서를 채워보자
```
1 2 6 7 15 16 28 
3 5 8 14 17 27 
4 9 13 18 26 
10 12 19 25 
11 20 24 
21 23 
22
```

- 이렇게 채워질 것
- 이때의 행과 열을 순서대로 출력하면 됨
- 두 번째 대각선은 왼쪽 아래로 증가 -> 짝수 번째 대각선은 왼쪽 아래로 증가
- 세 번째 대각선은 오른쪽 위로 증가 -> 홀수 번째 대각선은 오른쪽 위로 증가
- n번째 대각선의 범위 -> S(n-1) + 1 ~ S(n)

### 💡 핵심 로직
- x가 주어지면? S(t)보다 처음으로 크거나 같아지는 t를 찾는다
- S(t) - x가 끝점에서 몇 번 이동해야하는지의 값
- n이 짝수면 정답은 n - 이동 / 1 + 이동 
- n이 홀수면 1 + 이동 / n - 이동
- S(n) = n(n+1)/2
- 시간복잡도 -> O(1)

### 📌 배운 점
- 수열의 합 공식으로 시간복잡도를 줄였습니다.
- 루프를 돌지 않고 수학으로 시간복잡도를 줄였습니다.
