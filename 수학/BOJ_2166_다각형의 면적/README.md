## 💡 BOJ 2166 - 다각형의 면적[[문제 링크](https://www.acmicpc.net/problem/2166)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `수학` |
| 사용 알고리즘 | |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 2차원 평면상에 N개의 점으로 이루어진 다각형이 있다
- 이 다각형의 면적을 구하라

**[제약 사항]**

- N <= 10,000
- 좌표값은 절댓값이 100,000을 넘지 않는 정수다
- 면적을 출력할 때에는 소수점 아래 둘째 자리에서 반올림하여 첫째 자리까지 출력한다

### 🔍 풀이 접근
- 다각형 내 삼각형의 넓이를 더해서 다각형 넓이를 구하면 되지 않을까?
- 벡터 외적으로 삼각형 넓이를 구하면 되잖슴

**[설계]**

- 0번째 점을 기준점으로 한다
- 1번째 점과 2번째 점으로 벡터를 만들어서 넓이를 계산한다
- 2번째 점과 3번째 점으로 벡터를 만들어서 넓이를 계산한다
- ...
- N-1번째 점과 N번째 점으로 벡터를 만들어서 넓이를 계산한다

**[오류]**

- 점이 뒤죽박죽이면, 넓이가 중복된다

### 💡 핵심 로직

- 외적을 확장하면 됨
- 외적은 부호 있는 넓이라서, 겹치는 영역이 고려됨
- 그래서 그냥 외적한거 쭉 더하면 넓이 나옴 ㅇㅇ

### 📌 배운 점
- 기하적인 상식을 바탕으로 문제 실마리를 찾는 연습을 했습니다.
- 다각형의 넓이를 구하는 방법을 알게 되었습니다.
