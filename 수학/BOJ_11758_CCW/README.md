## 💡 BOJ 11758 - CCW [[문제 링크](https://www.acmicpc.net/problem/11758)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `수학` |
| 사용 알고리즘 |  |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 2차원 좌표 평면 위에 있는 점 3개 P1, P2, P3가 주어진다
- P1, P2, P3를 순서대로 이은 선분이 어떤 방향을 이루고 있는지 구하라

**[제약 사항]**

- -10,000 <= x1,y1,x2,y2 <= 10,000
- P1, P2, P3의 좌표는 서로 다르다
- 반시계 방향을 나타내면 1, 시계 방향을 나타내면 -1, 일직선이면 0을 출력

### 🔍 풀이 접근

- 점을 순서대로 찍었을 때 어떤 방향으로 회전하는지를 구하는 문제다
- 세 개의 점이 만드는 두 개의 벡터를 외적했을 때, 부호를 출력하면 되는 문제 아닌가?

### 💡 핵심 로직

- P1P2 벡터와 P2P3 벡터를 만든다
- 둘을 외적한다
- 0이면 0출력, 음수면 -1출력, 양수면 1출력

### 📌 배운 점

- 외적의 정의를 이해하고 문제에 적용하는 연습을 했습니다.
