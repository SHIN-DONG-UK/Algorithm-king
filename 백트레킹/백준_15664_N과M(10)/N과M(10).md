# 1. 이해

```
[조건]
- N개의 자연수와 자연수 M이 주어짐
- N개의 자연수 중에서 M개를 고른 수열
- 고른 수열은 비내림차순
- N개의 자연수는 같은 수가 포함될 수 있음

[입력]
- N, M (1 ≤ M ≤ N ≤ 8)
- N개의 자연수

[출력]
- 조건에 맞는 수열 출력
- 사전순으로 증가
- 중복x
```

# 2. 설계

```
조합 + path값으로 숫자 배열 인덱싱 + 중복 제거를 위한 조건문 추가

[조건문]
- 같은 레벨에 사용된 숫자는 다시 쓰지 못하도록 visited해놓으면 됨
- 그리고 해당 레벨이 끝나면 visited 풀어주기
- visited[idx][10001] 선언 후 해당 층에서 사용한 숫자 방문 처리
- 해당 층 방문이 끝나면 memset으로 해당 층만 memset으로 초기화
```
