# 1. 이해

```
[조건]
- 자연수 N과 M이 주어짐
- 같은 수를 여러 번 골라도 됨
- 고른 수열은 비내림차순

[입력]
- 자연수 N과 M ( 1 ≤ M ≤ N 8 )

[출력]
조건을 만족하는 수열 한 줄씩
```
