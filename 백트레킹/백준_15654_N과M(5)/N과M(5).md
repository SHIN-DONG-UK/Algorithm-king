# 1. 이해

```
[조건]
- N개의 자연수와 자연수 M이 주어짐
- 조건을 만족하는 길이가 M인 수열을 모두 구하라
- N개의 자연수는 모두 다른 수임

[입력]
- N, M (1 ≤ M ≤ N ≤ 8)
- N개의 수

[출력]
- 조건을 만족하는 수열 출력
- 사전 순으로 증가해야 함
```

# 2. 설계

```
1. 앞에서 만든 순열구하는 코드를 가져온다
2. 그리고 구한 순열을 인덱스로 생각한다
3. path 배열에 저장된 인덱스로 N개의 자연수를 넣어 둔 num 배열에서 꺼내 쓴다
4. 사전 순으로 증가하도록 num 배열을 오름차순 정렬해 놓는다

[요약]
순열 코드로 num배열 인덱싱하고 출력
```
