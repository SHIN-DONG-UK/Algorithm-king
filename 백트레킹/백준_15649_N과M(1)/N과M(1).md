# 1. 이해

```
[입력]
- 자연수 N과 M이 주어짐
- 1≤M≤N≤8

[출력]
- 1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열 출력
```

# 2. 설계

```
[순열]
1. path 배열과 visited 배열을 준비한다
2. N개의 노드를 M의 깊이만큼 탐색한다
		- go()함수는 어떤 깊이에서 N개의 노드를 방문하는 함수이다
		- 따라서 N번 반복하고, 다음 깊이로 넘어가는 재귀 호출이 존재한다
3. 다음 깊이를 방문할 때, path상의 노드는 방문하지 않는다고 해보자
4. 그럼 순열을 구할 수 있다
5. 다음 깊이로 들어가기 전에 해당 노드를 방문처리 하면, 다음 깊이에서 방문할 수 없다
6. 다음 깊이를 방문하고 돌아왔을때는 방문처리를 풀어주어 다음 loop에서 사용할 수 있도록 한다
```
