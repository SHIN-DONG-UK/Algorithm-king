# 1. 이해

```
- 팀을 만들자
- 팀 인원수 제한x
- 단 한 명만 같이하고 싶은 사람으로 지명크(자기자신 가능)
- 학생이 다른 학생을 가리켰을 때, 만약 closed 되면 팀이다

[출력]
- 어느 팀에도 속하지 않은 학생 수를 구하라

[입력]
- 테스트 케이스 T
- 학생 수 n
- 선택된 학생들의 번호 n개
```
