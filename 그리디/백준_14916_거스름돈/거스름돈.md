# 1. 이해

```
[상황]
- 2원짜리 동전, 5원짜리 동전이 있음
- 동전의 개수가 최소가 되도록 거슬러 주자

[출력]
- 거스름돈이 n일 때, 동전 개수 최소를 구하라
- 거슬러 줄 수 없으면 -1을 출력

[입력]
- n (1≤n≤100000)
```

# 2. 설계

```
- 일단 2와 5로 모든 수를 표현할 수 없으니까, 분명히 거슬러줄 수 없는 돈이 있음
- 일단 5원으로 다 거슬러 줘
- 그리고 나머지 2원으로 거슬러
- 안돼? 그럼 5원 까고 2원으로 줘
- 5원부터해서 2원까지 간 다음에, 가능한 애들 중 최소를 구하면 되잖아

[정리]
1. 5원 올인
2. 5원 하나 빼고 2원 채워 -> 반복
	- 가능하면 동전 개수 업데이트
3. 가능한 경우가 없으면 -1
```
