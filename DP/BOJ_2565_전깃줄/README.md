## 💡 BOJ 2565 - 전깃줄 [[문제 링크](https://www.acmicpc.net/problem/2565)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `DP` |
| 사용 알고리즘 | |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 두 전봇대 A와 B 사이에 전깃줄을 추가하는데, 서로 교차하는 경우가 발생
- 이들 중 몇 개의 전깃줄을 없애 전깃줄이 교차하지 않도록 만들려고 함
- 전깃줄의 개수와 연결된 번호가 주어질 때, 남아있는 모든 전깃줄이 서로 교차하지 않게 하기 위해 없애야 하는 전깃줄의 최소 개수를 구하라

**[제약 사항]**

- 전깃줄의 개수 <= 100
- 전깃줄 위치의 번호 <= 500
- 같은 위치에는 하나의 전깃줄만 연결됨

### 🔍 풀이 접근
- A를 기준으로 오름차순 정렬해보자
- 전깃줄이 꼬이지 않기 위해, B가 계속해서 증가해야 한다
- 그럼 최장 증가 부분 수열을 찾는 문제와 동일해진다
- 예제의 경우, 8 2 9 1 4 6 7 10 -> 5 (정답 8 - 5 = 3)

### 💡 핵심 로직
- dp[i] 는 i번 째를 마지막으로 했을 때 부분 수열의 길이다
- 0번 index부터 i-1까지 돌면서, 연결할 수 있는 애들 중 가장 큰 값을 찾아 +1한다
