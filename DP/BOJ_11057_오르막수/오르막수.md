# 1. 이해

```
[오르막 수]
- 오르막 수는 수의 자리가 오름차순을 이루는 수
- 2234, 3678, 11119는 오르막 수
- 2232, 3676, 91111은 오르막 수가 아님
- 0으로 시작 가능

[출력]
- 수의 길이 N이 주어질 때, 오르막 수의 개수를 구하라

[입력]
- N, 1 ≤ N ≤ 1000
```
