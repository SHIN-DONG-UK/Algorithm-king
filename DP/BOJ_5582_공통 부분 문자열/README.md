## 💡 BOJ 5582 - 공통 부분 문자열 [[문제 링크](https://www.acmicpc.net/problem/5582)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `DP` |
| 사용 알고리즘 | |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 두 문자열이 주어졌을 때, 두 문자열에 모두 포함된 가장 긴 공통 부문 문자열을 찾아라
- 부분 문자열이란, 연속한 문자열을 말함

**[제약 사항]**

- 1 <= 문자열 길이 <= 4,000

### 🔍 풀이 접근
- 최장 공통 부분 문자열이라는 카테고리임
- LCS 알고리즘 문제임

### 💡 핵심 로직
- 같으면 대각선 + 1
- 다르면 0

### 📌 배운 점
- 최장 공통 부분 수열과 최장 공통 부분 문자열의 차이를 이해했습니다.
- DP적 사고를 다시 한 번 상기했습니다.
