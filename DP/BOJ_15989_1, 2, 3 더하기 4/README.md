## 💡 BOJ 15989 - 1,2,3 더하기 4 [[문제 링크](https://www.acmicpc.net/problem/15989)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `DP` |
| 사용 알고리즘 | |
| 난이도 | `골드5` |

### ✅ 문제 설명

**[문제 요약]**

- 정수 n이 주어졌을 때, n을 1, 2, 3의 합으로 나타내는 방법의 수를 구하라
- 단, 합을 이루고 있는 수의 순서만 다른 것은 같은 것으로 친다

**[제약 사항]**

- 1 <= n <= 10,000

### 🔍 풀이 접근

- dp로 풀어보자

### 💡 핵심 로직

- 경우의 수가 중복되지 않게 누적되기 위해, 사용하는 수의 종류를 통제해야 한다
- 따라서 2차원 dp로 각 수에 대해 사용할 수 있는 수를 제한하도록 한다

### 📌 배운 점
- 저번 문제에서 익힌 '사용할 수 있는 수의 종류를 통제하기 위한 2차원 dp'에 대해 학습한걸 적용했습니다.
