## 💡 BOJ 1351 - 무한 수열 [[문제 링크](https://www.acmicpc.net/problem/1351)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `DP` |
| 사용 알고리즘 | `메모이제이션` | 
| 난이도 | `골드5` |

### ✅ 문제 설명

**[문제 요약]**

- 무한 수열 A는 다음과 같다
   - A0 = 1
   - Ai = A⌊i/P⌋ + A⌊i/Q⌋ (i >= 1)

- N, P와 Q가 주어질 때, A_N을 구하라

**[제약 사항]**

- ⌊x⌋는 x를 넘지 않는 가장 큰 정수
- 0 <= N <= 10^12
- 2 <= P, Q <= 10^9

### 🔍 풀이 접근

- 7 2 3을 해보자
- A7을 구해야 한다
- A0 = 1
- A1 = A⌊1/2⌋ + A⌊1/3⌋ = A0 + A0 = 2
- A2 = A⌊2/2⌋ + A⌊2/3⌋ = A1 + A0 = 3
- A3 = A⌊3/2⌋ + A⌊3/3⌋ = A1 + A1 = 4
- A4 = A⌊4/2⌋ + A⌊4/3⌋ = A2 + A1 = 5
- A5 = A⌊5/2⌋ + A⌊5/3⌋ = A2 + A1 = 5
- A6 = A⌊6/2⌋ + A⌊6/3⌋ = A3 + A2 = 7
- A7 = A⌊7/2⌋ + A⌊7/3⌋ = A3 + A2 = 7
- 입력 N에 대해 1부터 N까지 순서대로 구하면 될 것같지만, N이 최악의 경우 10^12이기 떄문에 반드시 TLE가 발생한다
- 따라서 Ax의 값을 순차적으로 구해서는 안된다
- 구할 필요가 없는 애들은 넘기는 식으로 해야 한다
- Top-Down방식으로 문제를 해결해보자
- AN = A⌊N/P⌋ + A⌊N/Q⌋에서 A⌊N/P⌋를 먼저 구해보자
- 그리고 이미 구한적 있으면 더 이상 구할 필요는 없다
- 구하지 않았으면 계속 분기에 들어간다
- 이런식으로 가지치기를 하면 문제를 해결할 수 있지 않을까?


### 💡 핵심 로직

- Top-Down 재귀를 수행한다
- memoization을 통해 시간복잡도를 줄인다
