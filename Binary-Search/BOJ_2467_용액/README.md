### ✅ 문제 설명
**[문제 요약]**

- 많은 종류의 산성 용액과 알칼리성 용액이 있다
- 각 용액에는 그 용액의 특성을 나타내는 하나의 정수가 부여된다
- 산성 용액은 1~10억
- 알칼리성 용액은 -10억~-1
- 같은 양의 두 용액을 혼합한 용액의 특성값은 두 용액 특성값의 합
- 이 연구소에서는 같은 양의 두 용액을 혼합하여 특성값이 0에 가까운 용액을 만들려고 함
- 두 알칼리성 혹은 두 산성이 정답일 수도 있음
- 산성 용액과 알칼리성 용액의 특성값이 정렬된 순서로 주어졌을 때, 이 중 두 개의 서로 다른 용액을 혼합하여 특성값이 0에 가장 가까운 용액을 만들어내는 두 용액을 찾아라

**[제약 사항]**

- 전체 용액의 수 N <= 100,000
- N개의 용액들의 특성값은 서로 다르다

### 🔍 풀이 접근
- 2개를 선택하는 거니까, NC2를 하면 O(N^2)
- 그럼 TLE
- 순차적으로 찾을 필요가 있나? 이분탐색으로 찾으면 되지 않을까?
- 그니까 지금 arr[i]=-99라고 하자
- 99를 찾는게 가장 이상적이니까, 99를 찾아보자
- 근데 99가 없을 수도 있잖아? 그럼 마지막으로 찾은 애 근처가 가장 최적임
-99 -2 -1 4 33 98 200

- 그래서 마지막으로 찾은 애랑 i이랑 더해서 갱신하면 되잖슴
- 그럼 arr[i]이 양수이면? -> left = 0 ~ right = i-1 이렇게 잡아야겠지

### 💡 핵심 로직
1. outer loop에서 모든 숫자를 돈다
2. 이분탐색으로 최선을 찾는다  
i < 0이면, left = i + 1, right = N - 1  
i > 0이면, left = 0, right = i - 1
3. 두 숫자의 합의 절댓값을 글로벌 최적과 비교해서, 더 작으면 글로벌 최적을 갱신하고 두 용액도 갱신한다
