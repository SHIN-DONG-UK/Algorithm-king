## 💡 BOJ 15591 - MooTube [[문제 링크](https://www.acmicpc.net/problem/15591)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Tree` |
| 사용 알고리즘 | `DFS` |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 1부터 N까지 번호가 붙여진 동영상이 있다
- 각 동영상은 정점, USADO는 엣지
- 어떤 동영상에 대해, 값 K를 정해서 그 동영상과 USADO가 K이상인 모든 동영상이 추천되도록 하고싶다
- K값이 주어질 때, 추천 동영상의 개수를 출력하라

**[제약 사항]**

- 동영상 번호 N <= 5,000
- 질문 개수 Q <= 5,000
- USADO r <= 10억

### 🔍 풀이 접근

- 문제를 추상화하면, 동영상끼리 트리 구조로 구성되어 있다
- 또 모든 정점을 방문하면서 각 정점에 방문하는 USADO를 전부 구해도 된다
- 전부 방문하면서 USADO가 K 이상이면 카운트하는식으로 구현해보자


### 💡 핵심 로직

- 주어지는 입력을 인접 리스트 방식으로 구성한다
- 다른 정점으로 이동할 때마다 USADO를 업데이트한다
- 업데이트한 USADO를 바탕으로 K 이상이면 카운트한다
- DFS로 구현한다

### 📌 배운 점

- 문제를 추상화하여 트리 구조를 활용했으며, 적절한 알고리즘을 선정해 문제를 해결하는 연습을 했습니다.
