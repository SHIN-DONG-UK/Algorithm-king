# 1. 이해

```
- M x N 모눈종이
- M, N 그리고 K개의 직사각형 좌표가 주어질 떄, 분리된 영역의 개수와 넓이를 구하라

[입력]
- M N K
- K개의 사각형
	- 왼쪽 아래 꼭짓점, 오른쪽 위 꼭짓점
	- a b c d라고 하면
		y : a -> c
		x : b -> d
		for문으로 방문처리하면 됨
```

# 2. 설계

```
1. 맵을 돌면서 시작할 수 있는 칸을 찾는다
	- 시작할 수 있는 칸 == 방문처리되지 않은 칸
2. 그 칸에서 bfs 탐색한다
	- 상하좌우로 움직이면서 갈 수 있는 모든 노드를 방문하고, 방문처리까지 해줌
	- 한 번의 bfs 탐색이 끝나면 하나의 영역을 찾고 표시를 한 것임
3. bfs 탐색을 진행한 횟수를 정답으로 출력한다.
	- 각 덩어리의 크기를 계산하고 vector에 저장해 놓는다
	- 덩어리 크기는 queue에 들어간 노드의 개수와 같다
```
