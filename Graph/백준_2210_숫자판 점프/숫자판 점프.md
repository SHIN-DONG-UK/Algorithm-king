# 1. 이해

```
[맵]
- 5 x 5 숫자판
- 각 위치에 0~9의 수

[이동]
- 인접한 네 방향으로 다섯 번 이동 -> 6자리 수 생성
- 한 번 거쳤던 칸을 다시 거쳐도 됨
- 0으로 시작해도 됨

[출력]
- 숫자판이 주어졌을 때, 만들 수 있는 서로 다른 여섯 자리 수의 개수

[입력]
맵
```

# 2. 설계

```
1. 맵을 입력받는다
2. 모든 위치에서 dfs를 시작해본다
3. DAT를 만들어서 중복을 제거한다
	- 최대 여섯 자리 수이므로 999999크기의 배열을 만들어 놓는다
	- 숫자를 만들고 DAT배열에 카운팅한다
	- 나중에 출력할 때는 인덱스만 출력해준다
	- 또는, map으로 정답배열을 구성한다
```
