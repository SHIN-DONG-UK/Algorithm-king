# 1. 이해

```
[조건]
- 출발 : 상근이네 집
- 맥주 : 한 박스(맥주 20개)
- 50미터에 한 병씩 마신다(50미터를 가려면 무조건 한 병 마셔야 함)
- 맥주 추가 구매 가능
- 그러나 박스에 들어있는 맥주는 20병을 넘을 수 없음
- *편의점을 나선 직후에도 50미터를 가기 전에 맥주 한 병을 마셔야 함

[출력]
- 편의점, 상근이네 집, 펜타포트 락 페스티벌 좌표가 주어질 때, 상근이와 친구들이 행복하게
페스티벌에 도착할 수 있는지 구하라
- 페스티벌에 도착할 수 있으면 happy, 중간에 맥주가 바닥나서 더 이상 이동할 수 없으면 sad

[입력]
- 테스트 케이스 T MAX_T 50
- 맥주를 파는 편의점의 개수 N MAX_N 100(0가능)
- 상근이네 집 좌표
- 편의점 좌표
- 펜타포트 락 페스티벌 좌표
- x, y(-32768 <= x, y <= 32767)
```

# 2. 설계

```
[설계]
- 그래프를 구성해야 한다
- 문제를 읽어보면, 한 번에 1000m를 갈 수 있다는 뜻으로 해석할 수 있다
- 즉, 노드와 노드사이의 맨해튼 거리가 1000이하이면 갈 수 있고, 초과이면 갈 수 없다
- 이렇게 시작점 -> 편의점
편의점 -> 편의점
편의점 -> 끝점
시작점 -> 끝점
의 그래프를 구성해 주고, 그냥 bfs돌리면 풀 수 있다
```
