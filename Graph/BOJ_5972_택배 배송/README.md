## 💡 BOJ 5972 - 불 [[문제 링크](https://www.acmicpc.net/problem/5972)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Graph` |
| 사용 알고리즘 | `다익스트라` |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 현서는 찬홍이에게 택배를 배달해야 한다
- 현서는 지나가는 소들에게 여물을 주는데, 최소한으로 주고 싶다
- N개의 헛간과 M개의 양방향 길이 있다
- 각 길은 C_i마리의 소가 있다
- 현서는 헛간 1에 있고, 찬홍이는 헛간 N에 있다
- 현서가 찬홍이에게 가는 길의 길이는 고려하지 않을 때, 여물을 주는 최소비용은 얼마?

**[제약 사항]**

- 1 <= N <= 50,000
- 1 <= M <= 50,000
- 1 <= C_i <= 1,000

### 🔍 풀이 접근

- Brute-Force로 문제를 해결한다고 해보자
- BFS 탐색을 수행하면, 시간복잡도는 O(NM)이므로 최악의 경우 TLE다
- BFS탐색을 수행할 때, 매 단계마다 최소한을 선택하면 결국 마지막에는 최적해가 나온다
- 다익스트라를 사용해보자

### 💡 핵심 로직

- best 배열을 선언한다
- best 배열을 무한대로 초기화한다
- 현재 노드에서 이웃한 노드로의 거리를 초기화한다
    - 만약 기존의 거리보다 더 적게 갈 수 있으면, PQ에 넣는다
- PQ에 더 이상 노드가 없으면, 반복을 종료한다
- best[N]의 값을 출력한다

### 📌 배운 점

- 다익스트라를 외워서 푸는게 아닌 최적화 측면을 고려해서 접근해보았습니다.
