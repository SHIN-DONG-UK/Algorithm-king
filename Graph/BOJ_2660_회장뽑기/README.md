## 💡 BOJ 2660 - 회장뽑기 [[문제 링크](https://www.acmicpc.net/problem/2660)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `그래프 탐색` |
| 사용 알고리즘 | `BFS` |
| 난이도 | `골드5` |

### ✅ 문제 설명

**[문제 요약]**

- 모임에서 회장을 선출하려고 한다
- 어느 회원이 다른 모든 회원과 친구이면, 1점
- 다른 모든 회원이 친구이거나 친구의 친구면, 2점
- 다른 모든 회원이 친구이거나, 친구의 친구이거나, 친구의 친구의 친구이면 3점
- 4점, 5점... 등도 같은 방법으로 정해짐
- 각 회원의 점수를 정할 때 어떤 두 회원이 친구사이이면서 동시에 친구의 친구사이이면, 이 두사람은 친구사이라고 본다
- 회장은 회원들 중에서 가장 작은 사람이 된다
- 회장의 점수와 회장이 될 수 있는 모든 사람을 출력하라

**[제약 사항]**

- 회원의 수는 50명 이하
- 마지막 줄에는 -1이 두 개 주어진다

### 🔍 풀이 접근

- 각 회원에서 시작해서 BFS 탐색을 수행하라는 문제로 보인다
- BFS 탐색을 수행했을 때 가장 먼 거리가 점수가 된다

### 💡 핵심 로직

- 인접리스트를 만든다
- dist 배열도 만든다
- 1번 노드부터 BFS 탐색을 수행한다
- dist를 기록한다
- N번 노드까지 반복한다
- 최솟값을 미리 갖고 있는다
- dist 배열에서 최솟값과 동일한 노드 번호 개수를 세고, 이를 동적 배열(vector)에 저장한다
- 출력한다

### 📌 배운 점
- 문제를 추상화하고 flood-fill을 적용하는 연습을 했습니다.
