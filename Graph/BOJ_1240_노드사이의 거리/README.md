## 💡 BOJ 1240 - 노드사이의 거리 [[문제 링크](https://www.acmicpc.net/problem/1240)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Graph` |
| 사용 알고리즘 | `BFS` |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- N개의 노드로 이루어진 트리가 주어지고 M개의 두 노드 쌍을 입력받을 때 두 노드 사이의 거리를 출력하라

**[제약 사항]**

- 2 <= N <= 1,000
- 1 <= M <= 1,000
- 트리 상에 연결된 두 점과 거리는 10,000이하 자연수
- 트리 노드의 번호는 1부터 N까지 자연수이며, 두 노드가 같은 번호를 갖는 경우는 없다

### 🔍 풀이 접근

- 트리라는 가정이 있으므로, 사이클이 발생하지 않는다
- Breadth First Traversal을 하면 된다

### 💡 핵심 로직

- 인접리스트 방식으로 트리를 구성한다
- BFS 탐색하는데, queue상에서 전달할 때 next와 dist값이 필요하다

### 📌 배운 점
- 트리에서 BFS하는 연습을 했습니다.
