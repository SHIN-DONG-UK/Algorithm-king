# 1. 이해

```
[문제]
- 공기와 접촉하는 치즈가 한 시간 마다 녹는다
- 치즈로 둘러싸인 공기와 접촉하는 치즈는 녹지 않는다

[출력]
- 공기 중에서 치즈가 모두 녹아 없어지는 데 걸리는 시간과
모두 녹기 한 시간 전에 남아있는 치즈조각이 놓여 있는 칸의
개수를 구하라

[입력]
- 세로 가로 (최대 100)
- 판
```
