## 💡 BOJ 2573 - 빙산 [[문제 링크](https://www.acmicpc.net/problem/2573)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `그래프 탐색` |
| 사용 알고리즘 | `BFS` |
| 난이도 | 골드5 |

### ✅ 문제 설명
- **R, G, B 각 그룹의 개수를 찾아라**

- 그런데, 적록색약은 R과 G를 구분할 수 없다 -> 둘을 같은 것으로 봐야 한다

### 🔍 풀이 접근
- `BFS`해서 덩어리를 인식한다

### 💡 핵심 로직
- R과 G를 같은 것으로 본다

### 📌 배운 점
- `BFS` 탐색의 포맷을 훈련했습니다.
