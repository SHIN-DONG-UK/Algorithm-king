## 💡 BOJ 15681 - 트리와 쿼리 [[문제 링크](https://www.acmicpc.net/problem/15681)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Graph` |
| 사용 알고리즘 | `DFS` |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 간선에 가중치와 방향성이 없는 임의의 루트 있는 트리가 주어졌을 때, 아래의 쿼리에 답해보자
- 정점 U를 루트로 하는 서브트리에 속한 정점의 수를 출력한다

**[제약 사항]**

- 트리의 정점의 수 N과 루트의 번호 R, 쿼리의 수 Q가 주어진다
- 2 <= N <= 10^5
- 1 <= R <= N
- 1 <= Q <= 10^5

### 🔍 풀이 접근
- 걍 노드마다 하위 노드 개수 한 번 저장해놓고 쿼리 들어오면 저장한 값 출력하면 됨

### 💡 핵심 로직
- 인접리스트 방식으로 그래프를 구성한다
- 루트에서 DFS를 수행한다
- DFS 함수는 서브트리 개수를 리턴한다
- 왼쪽 서브트리 개수와 오른쪽 서브트리 개수를 현재 서브트리 개수로 저장한다
- 서브트리 개수 테이블을 참조하여 쿼리를 수행한다

### 📌 배운 점
- 트리에서 Deep First Traversal을 진행해 테이블을 만들고, 이 테이블에서 쿼리를 수행하도록 하는 트리 DP 문제를 풀이했습니다.
