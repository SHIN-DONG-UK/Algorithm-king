## 💡 BOJ 20364 - 부동산 다툼 [[문제 링크](https://www.acmicpc.net/problem/20364)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Tree` |
| 사용 알고리즘 | |
| 난이도 | `실버1` |

### ✅ 문제 설명
**[문제 요약]**

- 이진 트리 모양의 땅으로 이루어진 마을에 오리들이 살고 있다
- 땅 번호는 다음과 같이 매겨진다
   1. 루트 땅의 번호는 1이다
   2. 어떤 땅의 번호가 K라면, 왼쪽 자식 땅의 번호는 2K, 오른쪽 자식의 땅 번호는 2K + 1
- 오리들끼리 부동산 다툼이 있어서 해결책을 다음과 같이 제시함
   1. 오리들을 한 줄로 대기시킨다. 맨 처음 오리들은 1번 땅에 위치해 있다
   2. 오리들이 서있는 순서대로 원하는 땅을 가지도록 한다
- 만약 한 오리가 원하는 땅까지 가는 길에 이미 다른 오리가 점유한 땅이 있다면, 그 땅을 지나지 못해 해당 땅을 가질 수 없다 (원하는 땅까지 가는 길 = 가는 길 + 그 땅 포함)
- 해결책대로 땅 분배를 했을 때 각 오리별로 원하는 땅을 가질 수 있는지, 가질 수 없다면 처음 마주치는 점유된 땅의 번호를 구하라

**[제약 사항]**

- 땅의 개수 N <= 2^20
- 오리 수 Q <= 200,000

### 🔍 풀이 접근

- 완전 이진 트리에서 생각한다

### 💡 핵심 로직
- 땅 번호가 주어지면 2로 계속 나눠서 부모 노드로 이동한다
- 만약 현재 노드가 방문처리되었으면 그 번호로 갱신한다
- 루트에 도달했을 때, 
   - 한 번도 갱신되지 않았으면 땅 번호를 방문처리하고 0을 출력한다
   - 가장 마지막에 갱신된 번호를 출력한다

### 📌 배운 점
- 완전 이진 트리에서 자식에서 부모로의 이동을 활용해 문제를 풀었습니다.
