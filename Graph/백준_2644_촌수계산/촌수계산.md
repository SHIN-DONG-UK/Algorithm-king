# 1. 이해

```
- 부모와 자식사이 1촌

이거 하나로 촌수 계산 가능

[입력]
(첫 번째 줄)
사람 : n명 (1≤n≤100)

(두 번째 줄)
서로 다른 두 사람의 번호

(세 번째 줄)
부모 자식 관계수 : m개
.
.
.

[출력]
요구한 두 사람의 촌수를 출력하라
두 사람의 친척 관계가 없어 촌수가 없으면 -1 출력
```

# 2. 설계

```
[why bfs?]
같은 lev에 있는 사람들은 같은 촌수이므로 한번에 방문해야 함

[로직]
1. 그래프를 구성한다
2. start 노드에서 bfs를 시작해서 end 노드를 만날 때까지 while Loop을 돈다
3. end 노드를 만났으면 모든 노드를 방문하기 전에 종료한다 (return true)
	-> ans 출력
4. 모든 노드를 방문했는데 end 노드가 없다 (return false)
	-> -1 출력
```
