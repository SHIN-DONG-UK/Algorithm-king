## 💡 BOJ 17836 - 공주님을 구해라! [[문제 링크](https://www.acmicpc.net/problem/17836)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `그래프 탐색` |
| 사용 알고리즘 | `BFS` |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 용사는 공주님을 구하기 위해 N x M 크기의 성 입구 (1, 1)로 들어왔다
- 마법 벽을 피해 (N ,M) 위치에 있는 공주님을 구출해야 한다
- 공주는 T시간 이내로 용사를 만나지 못하면 죽는다
- 성에는 이전 용사가 사용하던 "그람"이 숨겨져 있다
- "그람"이 있으면 벽을 부수고 이동할 수 있다
- "그람"은 성 안에 반드시 한 개 존재한다
- 구출할 수 있다면 시간을 출력하고, 구출할 수 없으면 Fail을 출력한다

**[제약 사항]**

- 3 <= N, M <= 100
- 1 <= T <= 10,000
- 맵 정보에서, 0은 빈 공간, 1은 마법의 벽, 2는 그람
- (1, 1)과 (N, M)은 0이다

### 🔍 풀이 접근

- 단순한 BFS에서 '상태'가 추가된 버전으로 이해된다
- BFS 탐색을 하다가, "그람"을 만나면 (N, M)에 단순 계산으로 시간을 구할 수 있다
- 그리고 정상 BFS 탐색도 (N, M)에 도착한다
- 둘 중 더 작은 값이 업데이트 되도록 하면 된다

### 💡 핵심 로직

- BFS 탐색을 하는데, 만약 "그람"을 만나면 맨해튼 거리로 (N, M)에 시간을 업데이트 한다
- 만약 맨해튼 거리가 더 작으면, (N, M)을 업데이트한다

### 📌 배운 점
- 단순 BFS 탐색에 조건문을 분기해 정답을 판정하는 문제를 해결했습니다.
- 조건문을 통해 다양한 케이스를 고려하는 연습을 했습니다.
