## 💡 BOJ 2589 - 보물섬 [[문제 링크](https://www.acmicpc.net/problem/2589)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Graph` |
| 사용 알고리즘 | `BFS` |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 2차원 격자에 육지와 바다가 있음
- 상하좌우 이웃한 육지로만 이동 가능
- 보물은 서로 최단 거리로 이동했을 때 가장 긴 시간이 걸리는 육지 두 곳에 나뉘어 있음
- 한 칸 이동하는데 한 시간 걸림
- 2차원 격자가 주어지면, 보물이 묻혀 있는 두 곳 간의 시간을 구하라

**[제약 사항]**

- 2차원 격자의 가로, 세로 크기 <= 50

### 🔍 풀이 접근
- 육지가 바다로 나뉘어져서 여러 군집을 이룰 수 있다
- 따라서 Brute-Force하게 모든 2차원 격자를 검사해야 한다
- 군집 내에서도 위치에 따라 결과가 다르다
- 걍 모든 육지에서 BFS해서 최댓값을 구하면 되겠다

### 💡 핵심 로직
- 2차원 격자를 순회한다
- 바다인 경우 넘어간다
- 현재 위치에서 BFS를 수행해, 가장 멀리 이동한 값을 구한다
- 글로벌 최적해와 비교해서 갱신한다

### 📌 배운 점
- Brute-Force하게 BFS를 수행했습니다.
