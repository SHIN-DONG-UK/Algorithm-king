## 💡 BOJ 17609 - 회문[[문제 링크](https://www.acmicpc.net/problem/17609)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `문자열` |
| 사용 자료구조 | |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 한 문자를 삭제하여 회문으로 만들 수 있는 문자열 == 유사회문
- 문자열을 입력받아 회문인지, 유사회문인지, 일반 문자열인지 판단하라

**[제약 사항]**

- T개의 테스트케이스로 이루어짐
- T <= 30
- 문자열의 길이 <= 100,000
- 알파벳 소문자로만 이루어짐

### 🔍 풀이 접근

- 유사회문을 검사할 때는 그리디하게 검사하면 된다

### 💡 핵심 로직

- 왼쪽 끝과 오른쪽 끝에서 시작한다
- 서로 다른게 없으면 회문이다
- 다른게 나왔으면, 왼쪽을 삭제해본다
- 이후 다른게 등장하지 않으면, 유사 회문이다
- 다시 오른쪽을 삭제한다
- 이후 다른게 등장하지 않았으면, 유사 회문이다
- 다른게 또 등장하면 여기서 일반 문자열로 판정

### 📌 배운 점
- 문자열을 두 개의 포인터로 돌면서 예외가 없도록 조건문을 작성하는 연습을 했습니다.
