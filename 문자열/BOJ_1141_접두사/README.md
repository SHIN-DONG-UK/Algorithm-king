## 💡 BOJ 1141 - 접두사[[문제 링크](https://www.acmicpc.net/problem/1141)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `문자열` |
| 사용 자료구조 | `트라이` |
| 난이도 | `실버1` |

### ✅ 문제 설명
**[문제 요약]**

- 접두사 X 집합이란 집합의 한 단어가, 다른 단어의 접두어가 되지 않는 집합
- 단어 N개로 이루어진 집합이 주어질 때, 접두사X 집합인 부분집합의 최대 크기를 출력

**[제약 사항]**

- 단어 개수 N <= 50
- 단어는 알파벳 소문자로만 이루어짐
- 단어 길이 <= 50
- 집합에 같은 단어가 여러 개 있을 수 있음

### 🔍 풀이 접근
- 접두사가 나오면 트라이를 생각해볼 수 있다
- 트라이를 만들면 문제를 쉽게 풀 수 있지 않나?
- 왜냐하면 접두사X 집합인 부분집합의 최대 크기이므로 여러 개의 부분집합을 고려해야 한다
- 트라이를 만들면 쉽게 부분집합을 알 수 있지 않나?

### 💡 핵심 로직
- 트라이 리프 노드 개수가 최댓값이다

### 📌 배운 점
- 접두사 문제에서 트라이를 활용해 해결했습니다.
