## 💡 BOJ 25206 - 너의 평점은[[문제 링크](https://www.acmicpc.net/problem/25206)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `문자열` |
| 사용 자료구조 | |
| 난이도 | `실버5` |

### ✅ 문제 설명
**[문제 요약]**

- 전공 평점을 계산해주는 프로그램을 작성하라
- 전공 평점은 전공과목별 학점*과목평점의 합을 학점의 총합으로 나눈 값이다
- P/F 과목의 경우 등급이 P또는 F로 표시되는데, 등급이 P인 과목은 계산에서 제외해야 한다

**[제약 사항]**

- 20줄에 걸쳐 지훈이가 수강한 전공과목의 과목명, 학점, 등급이 공백으로 구분되어 주어진다
- 정답과의 절대오차 또는 상대오차가 10^-4이하이면 정답으로 인정
- 과목명의 길이 <= 50
- 과목명은 띄어쓰기 없음
- 학점은 1.0 2.0 3.0 4.0 중 하나
- 등급은 A+, A0, B+, B0, C+, C0, D+, D0, F, P 중 하나
- 적어도 한 과목은 등급이 P가 아님이 보장됨 -> 0으로 나누는 일은 없다

### 🔍 풀이 접근

- 과목명은 의미없는듯

### 💡 핵심 로직
- 학점을 누적해놓는다 (sum1)
- 한 과목마다 학점 * 평점해서 누적해놓는다 -> 단, P는 누적하지 않는다 (sum2)
- sum2 / sum1
