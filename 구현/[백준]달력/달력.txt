# 1. 이해

```
[코팅지]
1. 연속된 두 일자에 각각 일정이 1개 이상 있다면 이를 일정이 연속되었다고 표현
2. 연속된 모든 일정은 하나의 직사각형에 포함되어야 함
3. 연속된 일정을 모두 감싸는 가장 작은 직사각형의 크기만큼 코팅지를 오린다

[달력]
1. 일정은 시작날짜와 종료날짜를 포함
2. 시작일이 가장 앞선 일정부터 차례대로 채워짐
3. 시작일이 같을 경우 일정의 기간이 가장 긴 것이 먼저 채워짐
4. 일정은 가능한 최 상단에 배치됨
5. 일정 하나의 세로 길이는 1
6. 하루의 폭은 1

[출력]
코팅지의 면적을 출력하라
```
# 2. 설명

```
[구현]
1. DAT 배열 : 해당 일자에 일정의 개수를 담는 배열
2. 가로 : 연속성을 검사한다. DAT 배열에서 0이 아닌 구간을 탐색한다
3. 세로 : 연속한 구간에서 일정의 개수가 가장 큰 값 -> DAT 배열을 순회하면 얻을 수 있음
4. 배열 순회 : 시작과 끝 사이에서 순회하면서 코팅지 면적 계산
배열값이 0이면 start를 업데이트하고 배열값이 0이 아닐때까지 i++하면서 start찾기
끝+1까지 탐색할 수 있어야 길이를 구할 수 있음
```
