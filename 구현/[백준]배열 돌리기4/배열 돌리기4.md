# 1. 이해

```
[맵]
	- N x M (3<=N, M<=50)
	- 배열의 값 = 각 행에 있는 모든 수의 합 중 최솟값

[회전]
	- (r, c, s) -> (r-s, c-s) ~ (r+s, c+s)까지의 직사각형 범위
	- 가장 바깥쪽 테두리부터 안쪽으로 들어가면서 회전을 수행

[요구사항]
	- 배열 A와 사용 가능한 회전 연산이 주어졌을 때, 배열 A의 값의 최솟값을 구하라
	- 회전 연산은 모두 한 번씩 사용해야 함
	- 순서는 임의로 정해도 됨
	
-------------------------------------------------------------------------
[구현]
	1. 회전을 구현한다
		빡구현임. 반복문과 조건문을 잘 써서 구현할 수 있다
	2. 모든 경우의 수를 따진다.
		순열을 구해서 회전 연산을 모두 수행해본다
```

# 2. 설명

```
1. 회전
	- tMAP에 회전 연산된 값을 저장해 놓고, 원래 MAP에 덮어씌운다
	- 하나의 회전 연산은 s번의 회전이 수행된다
	- 각 회전의 시작 좌표는 회전 연산의 왼쪽위 꼭짓점 (r-s, c-s)에, 각각 k를 더한 값이 된다
	- 초기에 오른쪽을 바라보도록 출발하여, 현재 좌표의 MAP값을 다음 좌표의 tMAP에 저장한다
	- 현재 좌표가 꼭짓점이라면, 방향을 바꾸어 다음 좌표를 계산한다

2. 모든 경우의 수
	- 재귀함수를 이용하여 순열을 구한다
	- 바닥에 도달했을 때, 구해진 순열을 바탕으로 회전하고 최솟값을 갱신한다
```
