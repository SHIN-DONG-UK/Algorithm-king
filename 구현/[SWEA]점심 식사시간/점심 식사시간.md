# 1. 이해

```
[맵]
- N x N 정사각형

- 1 : 계단

- 1보다 큰 정수 : 계단

[이동 완료 시간]
모든 사람이 아래층으로 이동이 완료된 시간
(이동 시간 = 계단 이동 시간 + 계단 내려가는 시간)

[이동 시간 정의]
1. 계단 입구 까지 이동 시간
	- t = |y1-y2| + |x1-x2|
	
2. 계단을 내려가는 시간
	- *계단에 도착하고 1분 후, 내려갈 수 있음
	
	- 최대 3명이 함께 이동 가능
	
	- 3명이 내려가고 있으면 대기
	
	- K분 후에 이동 완료
	
	
[출력]
이동 완료 시간의 최솟값을 구하라
```

# 2. 설계

```
[고려할 점]
- 더 먼 계단으로 가는게 더 빠를 수 있음
	=> 모든 경우의 수를 고려한다
	
- 계단은 무조건 두 개

- 사람의 수 10 이하

[Step1] 모든 경우의 수를 구하는 방법
- 각 사람을 계단1과 계단2에 배치한다

- 재귀함수 혹은 비트마스킹으로 구현 가능

- ex) 5명의 사람
	10110 -> 1,3,4번 사람은 계단1 / 2,5번 사람은 계단 2

[Step2] 계단을 내려가는 시뮬레이션 구현
- State 구조체 : 도착시간과 계단에 머무른 시간을 저장하기 위함

- 대기열을 vector<State> s1,s2로 하고, 계단을 queue<State> q1,q2로 한다

- 도착 순서로 정렬한다(vector에 구현하므로 내림차순 정렬)

- 현재 시간과 1분이상 차이나면 계단에 들어갈 수 있다

- 계단 queue에 3명이 들어가있으면 1분이상 차이나도 들어갈 수 없다

- 계단 qeuue에 들어간 사람은 계단에 머무른 시간이 계산된다

- 계단에서 머무른 시간이 K와 같으면 queue에서 뺀다

- 만약 모든 컨테이너가 비어있으면, 종료한다
```
