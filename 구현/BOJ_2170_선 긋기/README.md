## 💡 BOJ 2170 - 선 긋기[[문제 링크](https://www.acmicpc.net/problem/2170)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `구현` |
| 사용 알고리즘 |  |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 한 점에서 다른 점으로 직선을 그을거임
- 여러 번 그려진 곳은 한 번씩만 계산
- 그려진 선의 총 길이를 구하라

**[제약 사항]**

- 선을 그은 횟수 N <= 1,000,000

### 🔍 풀이 접근
- 범위가 -10억 ~ 10억이라서 1단위로 방문처리를 하는 방법은 TLE
- O(N) 복잡도로 문제를 해결하면 된다
- x를 기준으로 오름차순정렬하고, 간단한 로직을 넣으면 되지 않을까?

### 💡 핵심 로직
- x를 기준으로 오름차순한다
- 길이를 누적하는데, 처음에는 y-x를 누적한다
- 그리고 새로운 출발점 t는 y가 된다
- 새로운 출발점 t가 다음 x보다 크거나 같고 y보다 작으면 y-t를 누적한다
- x보다 크고 y보다 크면 누적하지 않고 t를 다음으로 넘긴다
- x보다 작으면 y-x를 누적하고, y를 새로운 t로 업데이트한다

### 📌 배운 점
- O(N)만에 해결할 수 있음을 이해하고, 구현 로직을 작성하는 연습을 했습니다.
