# 1. 이해

```
[자물쇠]
- 16진수
- 시계방향 회전
- 각 변에는 동일한 개수의 숫자
- 시계방향 순으로 높은 자리 숫자에 해당

[출력]
- 비밀번호 출력
- 숫자를 만들 수 있는 모든 수 중, K번째로 큰 수를 10진 수로 만든 수
```

# 2. 설명

```
[rotate]
- 모든 경우의 수를 구하기 위해, rotate를 구현해야 함
- nums를 거꾸로 돌면서 현재 인덱스에 이전 인덱스 값을 대입하면 됨
- 단, 마지막 인덱스를 저장해 놓고 첫 번째 인덱스에 마지막 인덱스를 대입해야 함

[slice]
- 한 변에 해당하는 문자열을 잘라야 함
- 한 변의 길이는 N/4임
- 2차 for문으로 slicing 가능
- 단, ans_arr에 해당 문자열이 존재하면 push_back하면 안됨
- 존재하는지 안하는지 확인은 N번 돌아서 확인하면 됨

[check]
- 내림차순으로 정렬
- 문자열을 그대로 내림차순 정렬하면 10진수로 변환한 내림차순과 순서 동일
- K번째 숫자를 10진수로 바꾸기
- 역방향으로 돌면서 sixteen을 거듭제곱
- 각 요소의 문자를 숫자로 변환하기 위해 ASCII값으로 조건 처리
```
