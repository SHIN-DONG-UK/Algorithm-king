# 1. 이해

```
[상황]
- N x N 맵
	'G' : 이동 가능
	'T' : 이동 불가
	'X' : 현재 위치
	'Y' : 목표 위치

- RC카 조종
	'A' : 앞으로 이동 => 나무가 있는 곳이나 맵 밖은 무시됨
	'L' : 현재 방향에서 반시계 90도 회전
	'R' : 현재 방향에서 시계 90도 회전
	
- RC카는 위를 바라보면서 시작
- 커맨드가 주어졌을 때, 목적지에 도달할 수 있는지 검사

[출력]
- 목적지에 도달할 수 있으면 1, 아니면 0을 출력
- 공백으로 구분
- 하나의 test case에 여러 개의 커맨드가 주어질 수 있음
```
