## 💡 BOJ 2941 - 크로아티아 알파벳[[문제 링크](https://www.acmicpc.net/problem/2941)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `구현` |
| 사용 알고리즘 |  |
| 난이도 | `실버5` |

### ✅ 문제 설명
**[문제 요약]**

- 알파벳으로 크로아티아 알파벳을 나타내려고 함
- 알파벳 - 크로아티아 알파벳 매칭이 주어짐
- 이거 보고 알파벳 단어가 주어졌을 때, 크로아티아 알파벳으로 몇 개인지 출력해보셈
- 단, 표에 없는 알파벳은 한 글자로 치면 됨
- 표에 dz는 d와 z로 분리된 걸로 절대 보지 않음

**[제약 사항]**

- 글자 길이 <= 100
- 알파벳 소문자와 '-', '='로만 이루어짐

### 🔍 풀이 접근
- 그리디하게 검사하면 된다

### 💡 핵심 로직
- 먼저 표를 검사한다
  - 2개 or 3개를 한 번에 보는거임
  - 있다? 그 만큼 인덱스를 넘김
  - 없다? 그냥 한 글자로 치고 다음 인덱스로 넘어간다

### 📌 배운 점
- 조건을 그리디하게 검사하고, 만약 이 조건을 만족하지 않으면 단순 처리를 하는 하드코딩 비슷한걸 경험했습니다.
