# 1. 이해

```
[올림픽 순위 결정]
1. 금메달 수가 더 많은 나라
2. 금메달 수가 같으면, 은메달 수가 더 많은 나라
3. 금, 은메달 수가 모두 같으면, 동메달 수가 더 많은 나라

[출력]
- 각 국가의 금, 은, 동메달 정보를 입력받아서, 어느 국가가 몇 등을
했는지 출력하라

[입력]
- 국가 수 N (max 1000)
- 등수를 알고싶은 국가 K
- 국가 번호 금 은 동
- 국가 번호 금 은 동
- ...
```

# 2. 설계

```
- 그냥 커스텀 소팅을 수행하면 될 것 같다
- 같은 등수인 나라들을 구분해야 하므로, for문 탐색하여 K국가를 찾고
앞으로 거슬러 올라가면서 몇 개의 같은(금은동 같은) 국가가 있는지 센다
- 같은 국가 수 만큼 빼면 등수가 나온다
```
