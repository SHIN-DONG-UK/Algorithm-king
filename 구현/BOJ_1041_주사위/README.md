## 💡 BOJ 1041 - 주사위[[문제 링크](https://www.acmicpc.net/problem/1041)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `구현` |
| 사용 알고리즘 |  |
| 난이도 | `골드5` |

### ✅ 문제 설명
**[문제 요약]**

- 주사위의 여섯 면에는 수가 쓰여 있다
- 주사위는 N^3개 있다
- 이를 적절히 회전시키고 쌓아서, N x N x N크기의 정육면체를 만들려고 한다
- 이 정육면체는 탁자위에 있으므로, 5개의 면만 보인다
- N과 주사위에 쓰여 있는 수가 주어질 때, 보이는 5개의 면에 쓰여 있는 수의 합의 최솟값을 출력하라

**[제약 사항]**

- 1 <= N <= 1,000,000
- 1 <= 주사위에 적힌 수 <= 50

### 🔍 풀이 접근
- 탐색을 해보자
- N == 1이면, 그냥 젤 큰 거 빼고 나머지 다 더하면 된다
- N == 2이면, 가장 작은 3개 4번 더하고, 가장 작은 2개 4번 더하면 된다
- N >= 3이면, 
   - 가장 작은 3개 4번 더한다
   - 가장 작은 2개 (N-2)  4번 더한다
   - 가장 작은 2개 (N-1)  4번 더한다
   - 가장 작은 1개 (N-2) * (N-1) 4번 더한다
   - 가장 작은 1개 (N-1) * (N-1) 더한다

### 💡 핵심 로직
- 가장 작은 2개와 가장 작은 3개를 구해야 한다
- 6C2, 6C3으로 각 15개 20개 나오는데, 모두 가능한 건 아니다
- 그래서 모든 경우의 수에서 되는 것 중에서 최솟값을 결정해야 한다
