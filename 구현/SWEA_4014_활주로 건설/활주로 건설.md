# 1. 이해

```
[상황]
- N x N 크기의 맵
- 각 칸의 숫자는 지형의 높이
- 활주로는 가로 또는 세로로 일직선이 되어야 함
- 활주로는 높이가 동일한 구간에서 건설이 가능
- 따라서 경사로를 설치해야만 활주로를 건설할 수 있음
- 경사로
	- 높이 : 1
	- 길이 : x
	
[출력]
- 경사로의 길이 x와 절벽지대의 높이 정보가 주어질 때,
활주로를 건설할 수 있는 경우의 수를 출력하라

[입력]
- Test Case T
- N (max 20) X (2 ~ 4)
- 맵
	- 경사로의 높이는 항상 1 X는 2 ~ 4
	- 지형 높이는 1 ~ 6
```

# 2. 설계

```
[알고리즘]
1. 가로로 쭉 해보고, 세로로 쭉 해보는 껍데기가 필요
2. 이제 각 경로마다, 경사로 설치해서 활주로가 가능한지 검사
	- 어떻게 검사해?
	- 만약 높이가 2차이나는 곳은 그냥 활주로가 불가능함
		=> 2차이가 나는 순간 다음으로 넘어가
	- 높이가 1차이인 곳에서 설치를 트라이하고, 설치 안되면 바로 다음 넘어가
	
```
