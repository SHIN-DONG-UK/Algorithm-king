# 1. 이해

```
[맵]
- 4x4, 6x6, 8x8
- 가운데 세팅 후 시작
	백 흑
	흑 백

[규칙]
- '놓을 돌'과 '자신의 돌' 사이에 상대 돌이 놓인 경우만 돌을 둘 수 있음
- 이때, '사이'에 있는 돌을 내 돌로 만들 수 있음
- '사이' == 세로/가로/대각선
```

# 2. 설계

```
1. 맵 세팅
	- (N/2, N/2) = 백
	- (N/2, N/2 + 1) = 흑
	- (N/2 + 1, N/2) = 흑
	- (N/2 + 1, N/2 + 1) = 백
	
2. 돌을 둘 때마다 가로 세로 대각선 검사
	- 돌을 둔다
	- 가로 세로 대각선으로 이동하면서 내 돌을 만날 때까지 전진한다
	- 전진할 때 일단 내 돌로 다 바꾼다
	- (if) 내 돌을 만났다 -> 거기서 바로 끝낸다
	- (else) 못만났다 -> 원복하고 돌아온다
```

# 3. 재귀함수 긴급탈출

```
go() 함수는 같은 돌을 만나면 즉시 종료하기 위해 bool을 리턴하도록 하였다
같은 돌을 만나지 못하고 기저 조건에서 false를 만나면, 원복하도록 코드를 작성하였다
이때, 원복 후 return false를 하지 않으면 아무것도 return하지 않으므로 컴파일 오류가 발생해야 하지만, 
visual studio에서는 컴파일 오류를 발생시키지 않고 런타임에서 그냥 함수를 종료해버린다
이 부분을 주의하자
다음의 예시 코드로 테스트 해볼 수 있다
```

```cpp
#include <iostream>

using namespace std;

bool go(int lev);

int main()
{
	go(0);
}

bool go(int lev)
{
	if (lev == 4) return true;
	else if (lev == 3) return false;

	cout << lev << ' ';
	if (go(lev + 1)) return true;
	cout << lev << ' ';
	//return false;
}
```
