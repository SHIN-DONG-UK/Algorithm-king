## 💡 BOJ 1722 - 순열의 순서[[문제 링크](https://www.acmicpc.net/problem/1722)]

### ✅ 문제 설명
**[문제 요약]**

- 1부터 N까지의 수를 임의로 배열한 순열은 총 N! 가지가 있다
- 임의의 순열은 정렬할 수 있는데, 작은 수가 먼저 오는 것이 앞 순서이다
- N이 주어지면, 두 개의 소문제 중 하나를 푼다
- k가 주어지면 k번째 순열을 구하고, 임의의 순열이 주어지면 이 순열이 몇 번째 순열인지를 출력하라

**[제약 사항]**

- 1 <= N <= 20
- 1인 경우 k를 입력받고, 2인 경우 임의의 순열을 나타내는 N개의 수를 입력받는다


### 🔍 풀이 접근

- 모든 순열을 만드는 시간복잡도는 O(N!)이다
- 따라서 Brute-Force하게 문제를 해결할 수 없다
- k번째 순열을 구하라는 소문제부터 생각해보자

- 숫자 하나가 고정되면, 그 뒤로 (N-1)!의 경우의 수가 존재한다

- depth=1일 때, 
1~(N-1)! <-> 1 / (N-1)!+1~2(N-1)! <-> 2 / ... / (N-1)(N-1)!+1~N! <-> N
- 이런식으로 개수와 숫자를 매칭할 수 있다

### 💡 핵심 로직

- 1!부터 20!까지 값을 미리 배열에 저장해놓는다
