### ✅ 문제 설명
**[문제 요약]**

- 지민이는 길이가 64cm인 막대를 가지고 있다
- 그런데 길이가 x인 막대가 가지고 싶어졌다
- 지민이는 원래 가지고 있던 막대를 더 작은 막대로 자른 다음, 풀로 붙여서 길이가 x인 막대를 만들려고 한다
- 막대를 아래와 같은 과정으로 자르려고 한다
  - 지민이가 가지고 있는 막대의 길이를 모두 더한다. 처음에는 64cm막대 하나만 갖고 있다. 이때, 합이 x보다 크다면 아래와 같은 과정을 반복한다
    - 가지고 있는 막대 중 길이가 가장 짧은 것을 절반으로 자른다
    - 만약, 위에서 자른 막대의 절반 중 하나를 버리고 남아있는 막대의 길이의 합이 X보다 크거나 같다면, 위에서 자른 막대의 절반 중 하나를 버린다.
  - 이제, 남아있는 모든 막대를 풀로 붙여서 X를 만든다

- X가 주어졌을 때, 위 과정을 거친다면, 몇 개의 막대를 풀로 붙여서 X를 만들 수 있는지 구하라

**[제약 사항]**

- X <= 64인 자연수

### 🔍 풀이 접근

- 그대로 구현해보자

### 💡 핵심 로직

- cnt = 1
- sum = 64
- a = 64

- a를 2로 나눔
- sum - a/2가 x보다 크거나 같다면, sum -= a/2를 하고 a/=2를 한다
- 그렇지 않으면 sum은 그대로 두고 a/=2 (cnt++)
- sum이 X일 때까지 위 과정을 반복
- cnt 출력
