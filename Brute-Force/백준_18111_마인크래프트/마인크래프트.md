# 1. 이해

```
[땅 고르기]
- N x M의 맵
- 각 칸은 0이상 256이하
- 왼쪽 위 좌표는 (0, 0)
- 땅의 높이를 일정하게 바꾸는 작업임
	1. 좌표 (i, j)의 가장 위에 있는 블록을 제거하여 인벤토리에 넣는다(제거)
		2초
	2. 인벤토리에서 블록 하나를 꺼내어 좌표 (i, j)의 가장 위에 있는 블록 위에 놓는다(추가)
		1초

[출력]
- 땅 고르기 작업에 걸리는 최소 시간과 그 경우 땅의 높이를 출력
- 처음 B개의 블록을 갖고 있음
- 시간이 같은게 여러개 있다면 높이가 가장 높은 거

[입력]
- N, M, B (1≤M,N≤500, 0≤B≤6천4백만)
- 맵
```

# 2. 설계

```
1. 높이를 정한다
	-> 최대한 빨리 땅을 고르기 위해 현재 땅의 높이의 최댓값과 최솟값 사이에서 땅을 골라야 함
2. 해당 높이를 다 해본다 -> 왜냐? 언제 최소가 될지 다 해보고는 모르잖슴?
3. 그 중 최솟값을 찾는다
```
