## 💡 BOJ 15656 - N과 M(7) [[문제 링크](https://www.acmicpc.net/problem/15656)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `백트레킹(Back-tracking)` |
| 사용 알고리즘 | `중복 순열` |
| 난이도 | 실버3 |

### ✅ 문제 설명
- N개의 자연수와 자연수 M이 주어진다

- N개의 자연수는 모두 다르다

- N개의 자연수 중에서 M개를 고른 수열

- 같은 수를 여러 번 골라도 됨

### 🔍 풀이 접근
- 순열에서 방문처리 빼면 중복 순열이 되고, 중복 순열을 활용하면 되는 문제

### 💡 핵심 로직
```cpp
void go(int idx){
    if(idx==M){
        for(int i=0; i<M; i++){
            cout << nums[a[i]] << ' ';
        }
        cout << '\n';
        return;
    }
    
    for(int i=0; i<N; i++){
        a[idx] = i;
        go(idx + 1);
    }
}

```

### 📌 배운 점
- 중복 순열을 인덱스로 활용해서 문제를 해결했습니다.
