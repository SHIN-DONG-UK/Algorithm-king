## 💡 BOJ 1711 - 직각삼각형 [[문제 링크](https://www.acmicpc.net/problem/1711)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Brute-Force` |
| 사용 알고리즘 | |
| 난이도 | `실버1` |

### ✅ 문제 설명
**[문제 요약]**

- 2차원 평면에 N개의 점이 주어져 있다
- 이 중에서 세점을 골랐을 때, 직각삼각형이 몇 개나 있는지 구하라

**[제약 사항]**

- 3 <= N <= 1,500
- 좌표값의 절댓값 <= 10억
- 모든 점의 좌표는 다르다

### 🔍 풀이 접근
- 직각삼각형이라고 판단하기 위해, 2가지가 필요
- 하나는 점이 3개일 것
- 다른 하나는 3개의 변 중 두 개가 서로 수직일 것
- 수직임을 쉽게 알 수 있는 방법은 '내적'
- 단순히 O(N^3)가 아니라 nC3을 하면 중복을 피할 수 있음
- 그리고 시간도 계산해보면 5억번 반복 => 5초 시간제한에 아슬아슬
- 해볼만하다

### 💡 핵심 로직
- nC3을 설계하면 된다
- 첫 번째 루프 => i=0 ~ N-2
- 두 번째 루프 => j=i+1 ~ N-1
- 세 번째 루프 => k=j+1 ~ N
- 루프 내에서는 i점을 기준점으로 한 벡터 두 개를 만든 다음 내적해서 0이면 카운트한다

### 📌 배운 점
- 문제 전체를 Brute-Force로 설계하고 내부 로직을 수학을 활용해 최적해를 구하는 문제를 풀이했습니다.
