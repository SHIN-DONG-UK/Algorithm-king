## 💡 BOJ 1436 - 영화감독 숌 [[문제 링크](https://www.acmicpc.net/problem/1436)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Brute-Force` |
| 사용 알고리즘 | `슬라이딩 윈도우` |
| 난이도 | `실버1` |

### ✅ 문제 설명
**[문제 요약]**

- a와 b로만 이루어진 문자열이 주어질 때, a를 모두 연속으로 만들기 위해서 필요한 교환의 회수를 최소로 하라
- 이 문자열은 원형이기 때문에, 처음과 끝은 서로 인접해 있다

**[제약 사항]**

- 문자열 길이 <= 1,000

### 🔍 풀이 접근
- 이 문제를 Brute-Force하게 접근할 수 있을까?
- 그니까 가능한 모든 a의 형태를 다 검사해보는거임

### 💡 핵심 로직
- a의 개수를 세고, 그 길이만큼의 윈도우를 모든 위치에서 해본다 == 모든 a의 형태를 검사하는 것
- 윈도우를 움직여보면서 b의 개수가 최소인 윈도우를 찾는다
- 그 최소 개수가 정답임

### 📌 배운 점
- 윈도우 크기를 정하고 Brute-Force하게 탐색한다는 문제 형식을 풀이했습니다.
