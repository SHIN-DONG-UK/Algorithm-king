## 💡 BOJ 7490 - 0 만들기 [[문제 링크](https://www.acmicpc.net/problem/7490)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Brute-Force` |
| 사용 알고리즘 |  |
| 난이도 | `골드5` |

### ✅ 문제 설명

**[문제 요약]**

- 1부터 N까지의 수를 오름차순으로 쓴 수열 1 2 3 ... N이 있다
- 그리고 '+'나 '-' 또는 ''공백을 숫자 사이에 삽입할 수 있다고 하자
- 이렇게 만든 수식의 값을 계산하고 그 결과가 0이 되는 모든 수식을 찾아라

**[제약 사항]**

- 테스트 케이스 개수가 주어진다 (<10)
- 3 <= N <= 9

### 🔍 풀이 접근

- N이 9이하라서 Brute-Force로 문제를 풀어도 충분하다

### 💡 핵심 로직

- 1부터 N까지 순서대로 level이라고 생각하자
- 항상 다음 level로 가면서 '+' or '-' or ' '를 모두 시도해본다
- 숫자는 항상 buffer에 넣어놓는다
- 만약 공백이면 buffer의 숫자만 조작한다
- 만약 +나 -이면 buffer의 숫자를 빼서 결과에 반영하고 buffer를 비운다

### 📌 배운 점
- 기본 brute-force 폼에서 변형하는 연습을 했습니다.
