# 1. 이해

```
[CCTV]
- 5가지 종류의 CCTV
	- 1번 : 한 쪽 방향
	- 2번 : 반대 방향 두 개
	- 3번 : 직각 방향 두 개
	- 4번 : 세 방향
	- 5번 : 상하좌우
- CCTV는 감시할 수 있는 방향의 모든 칸을 감시할 수 있음
- 벽은 통과할 수 없음
- CCTV가 감시할 수 없는 영역은 사각지대
- CCTV는 회전시킬 수 있는데, 항상 90도 방향으로 해야 함
- 감시하려고 하는 방향이 가로 또는 세로
	-> 5번 CCTV는 회전할 필요 없음
- CCTV는 CCTV를 통과할 수 있다

[출력]
- 사무실의 크기와 상태, 그리고 CCTV의 정보가 주어졌을 때,
CCTV의 방향을 적절히 정하여 사각 지대의 최소 크기를 구하라
```

# 2. 설계

```
[알고리즘]
- 위 CCTV에 따라 언제 최소 사각지대가 될 지 모르므로, 모든 경우를 탐색
하는 Brute Force 알고리즘으로 뼈대를 구성한다
- 맵에 있는 모든 CCTV 개수만큼 중복순열을 구해서 해당 방향으로 돌린다음
사각지대를 구해본다
- 최대 CCTV의 개수는 8이므로, 4^8이다
- 각 case(중복순열)에 대해 CCTV가 감시가능한 부분을 표시하고 표시되지 않은
영역의 개수를 구하면서 최소를 업데이트 한다
```
