## 💡 BOJ 21317 - 징검다리 건너기 [[문제 링크](https://www.acmicpc.net/problem/21317)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Brute-Force` |
| 사용 알고리즘 |  |
| 난이도 | `실버1` |

### ✅ 문제 설명
**[문제 요약]**

- N개의 돌이 있고, 마지막 돌 틈 사이에 산삼이 있음
- 돌과 돌 사이를 점프하면서 이동하는데, 점프의 종류는 3가지
   - +1 (작은 점프)
   - +2 (큰 점프)
   - +3 (매우 큰 점프)
- 각 점프를 할 때 에너지를 소비하는데, 이 때 작은 점프와 큰 점프 시 소비되는 에너지는 점프하는 돌의 번호마다 다름
- 매우 큰 점프는 단 한 번이고 돌 상관없이 k 에너지 소비
- 산삼을 얻기 위해 필요한 에너지의 최솟값을 구하라

**[제약 사항]**

- 돌의 개수 N <= 20
- 에너지 <= 5,000

### 🔍 풀이 접근
- 일단 문제의 크기가 작기 때문에, Brute-Force하게 접근해도 된다
- 작은 점프와 큰 점프 두 가지를 Brute-Force하게 해보는데 그 와중에 매우 큰 점프를 넣어야 한다
- 매우 큰 점프는 딱 한 번 해야 하기 때문에 bool 변수를 둬서 딱 한 번만하게해야 함

### 💡 핵심 로직
- 재귀함수를 설계하면 된다
- 종료조건 = 현재가 마지막 돌이면 종료한다
- 한 칸 이동하는 재귀
- 두 칸 이동하는 재귀
- 세 칸 이동하는 재귀 (단, 딱 한 번만 가능)

### 📌 배운 점
- 최적해를 구하기 위해 `Brute-Force`하게 접근하고, 문제 요구사항을 구현하는 해결력을 길렀습니다.
