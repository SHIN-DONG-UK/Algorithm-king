## 💡 BOJ 1543 - 문서 검색 [[문제 링크](https://www.acmicpc.net/problem/1543)]

| 항목 | 내용 |
|------|------|
| 문제 유형 | `Brute-Force` |
| 사용 알고리즘 | |
| 난이도 | `실버5` |

### ✅ 문제 설명
**[문제 요약]**

- 문서와 검색하려는 단어가 주어진다
- 그 단어가 최대 몇 번 중복되지 않게 등장하는지 구하라

**[제약 사항]**

- 문서의 길이 <= 2,500
- 검색어 <= 50
- 문서와 단어는 알파벳 소문자와 공백으로 이루어짐

### 🔍 풀이 접근

- 단순 구현하면 될 것같다

### 💡 핵심 로직

- 문서의 0번 인덱스부터 끝 인덱스까지 탐색한다
- 0번 인덱스부터 단어와 일치하는지 비교한다
- 현재 인덱스 i부터 단어 길이만큼 비교했을 때 일치하면, 다음 인덱스는 i + 단어 길이
- 일치하지 않으면, i++
- 이렇게 했을 때 동일한 개수를 카운팅
